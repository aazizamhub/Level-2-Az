1
Pendahuluan & Kegunaan Multiple Regression

Multiple linear regression menggunakan dua atau lebih independent variables untuk mendeskripsikan variasi dari dependent variable — berbeda dengan simple linear regression yang hanya menggunakan satu independent variable. Dengan pendekatan yang lebih kompleks ini, analis bisa membangun model yang lebih realistis karena dalam dunia investasi yang sesungguhnya, hampir tidak ada fenomena finansial yang hanya dipengaruhi satu faktor saja.

Kalau digunakan dengan benar, multiple regression bisa menghasilkan prediksi yang lebih akurat, konstruksi portfolio yang lebih baik, dan pemahaman yang lebih dalam tentang driver-driver return sekuritas. Namun kalau digunakan sembarangan, hasilnya bisa berupa spurious relationships (hubungan palsu), prediksi yang buruk, dan kesimpulan yang menyesatkan.

1.1 Tiga Kegunaan Utama Multiple Regression

Dalam praktik investasi, multiple regression digunakan untuk tiga tujuan utama:

  • Mengidentifikasi hubungan antar variabel. Misalnya, seorang portfolio manager ingin memahami bagaimana return saham-sahamnya dipengaruhi oleh serangkaian faktor — size effect, value effect, profitability, dan investment aggressiveness. Tujuannya adalah mengestimasi Fama-French five-factor model yang akan memberikan pemahaman tentang faktor-faktor yang paling penting dalam menggerakkan excess return saham tertentu.
  • Menguji teori yang sudah ada. Misalnya, seorang financial adviser ingin mengidentifikasi apakah variabel-variabel tertentu seperti financial leverage, profitability, revenue growth, dan perubahan market share dapat memprediksi apakah suatu perusahaan akan mengalami financial distress.
  • Membuat forecast. Misalnya, seorang analis ingin menguji pengaruh berbagai dimensi country risk — seperti stabilitas politik, kondisi ekonomi, dan pertimbangan ESG — terhadap equity returns di negara tersebut.
1.2 Proses Regression — Alur Kerja Analis

Proses membangun dan menggunakan regression model mengikuti alur yang sistematis. Analis harus membuat beberapa keputusan penting di sepanjang proses ini:

  • Langkah pertama: Apakah dependent variable bersifat kontinu (seperti return dalam %)? Kalau ya → gunakan multiple linear regression. Kalau tidak, atau bersifat diskrit/biner (misalnya: apakah perusahaan ini akan jadi target akuisisi atau tidak? — jawabannya 1 atau 0) → gunakan logistic regression.
  • Langkah kedua: Spesifikasi model — tentukan variabel apa yang masuk sebagai independent variables. Variabel-variabel ini bisa berupa variabel kontinu (seperti financial ratios) atau variabel diskrit (dummy variables yang mengindikasikan keanggotaan dalam suatu kelompok, misalnya sektor industri).
  • Langkah ketiga: Estimasi model menggunakan software (Excel, Python, R, SAS, Stata) dan analisis output-nya — pastikan asumsi-asumsi regression terpenuhi dan goodness-of-fit criteria terpenuhi.
  • Langkah keempat: Setelah model diuji dan out-of-sample performance-nya dianggap memuaskan, gunakan model tersebut untuk mengidentifikasi hubungan, menguji teori, atau membuat forecast.
💡 Catatan Penting

Software yang mengestimasi model (Python, R, Excel) bekerja otomatis — tugas utama analis bukanlah melakukan kalkulasi manual, melainkan: (1) menspesifikasikan model dengan benar, dan (2) menginterpretasikan output dengan tepat. Kedua hal inilah yang diuji di CFA Level II.

Knowledge Check 1.1

Sebagai analis junior, kamu diminta menentukan tindakan yang tepat untuk setiap situasi berikut:

(a) Dependent variable bernilai 1 jika perusahaan adalah merger target dan 0 jika bukan.

Gunakan logistic regression. Karena dependent variable bersifat diskrit/biner (hanya bernilai 0 atau 1), bukan kontinu, maka ordinary linear regression tidak tepat. Logistic regression dirancang khusus untuk dependent variable yang bersifat kategorikal.

(b) Analis mengestimasi model dengan lima independent variables, dan tidak ada satupun yang signifikan secara statistik.

Adjust the model and re-estimate. Kalau tidak ada variabel yang signifikan, kemungkinan ada masalah dengan spesifikasi model — mungkin salah pilih variabel, ada multicollinearity, atau memang hubungannya tidak linear. Perlu ditinjau ulang.

(c) Residual tidak tampak homoskedastis, sehingga melanggar asumsi regression.

Adjust the model and re-estimate. Heteroskedasticity (variance residual tidak konstan) adalah pelanggaran asumsi yang harus diperbaiki — bisa dengan weighted least squares (WLS) atau robust standard errors — sebelum model bisa digunakan untuk inferensi statistik yang valid.

(d) Semua asumsi regression terpenuhi, overall fit signifikan, dan ini adalah model terbaik dari semua kemungkinan model.

Gunakan model tersebut untuk analisis dan prediksi. Kalau semua kriteria sudah terpenuhi, model siap digunakan.
2
Dasar-Dasar Multiple Regression

Tujuan simple regression adalah menjelaskan variasi dependent variable Y menggunakan variasi satu independent variable X. Tujuan multiple regression sama persis, hanya saja kita menggunakan variasi dari sekumpulan independent variables X₁, X₂, ..., Xₖ.

Ingat bahwa variasi Y didefinisikan sebagai:

Variasi Y (Sum of Squares Total) Variasi Y = Σᵢ (Yᵢ − Ȳ)² di mana Ȳ adalah rata-rata Y dari seluruh n observasi
2.1 Persamaan Model

Persamaan simple regression adalah:

Yᵢ = b₀ + b₁Xᵢ + εᵢ, i = 1, 2, 3, ..., n

Ketika kita menambahkan independent variables tambahan untuk membantu menjelaskan variasi Y, kita mendapatkan persamaan multiple regression:

Persamaan Multiple Regression Yᵢ = b₀ + b₁X₁ᵢ + b₂X₂ᵢ + b₃X₃ᵢ + ... + bₖXₖᵢ + εᵢ i = 1, 2, 3, ..., n | k = jumlah independent variables | n harus lebih besar dari k

Dalam persamaan ini, suku-suku yang melibatkan k independent variables merupakan bagian deterministik dari model, sedangkan error term εᵢ adalah bagian stokastik/acak dari model.

2.2 Interpretasi Koefisien

b₀ (intercept): Nilai yang diharapkan dari Y apabila semua independent variables bernilai nol. Dalam konteks multiple regression, ini adalah titik referensi baseline — kalau semua faktor yang kita masukkan ke model sama dengan nol, berapa nilai Y yang kita ekspektasikan?

b₁ sampai bₖ (slope coefficients / partial regression coefficients): Ini adalah bagian paling krusial yang perlu dipahami dengan benar. Setiap slope coefficient bⱼ dalam multiple regression dikenal sebagai partial regression coefficient atau partial slope coefficient, dan harus diinterpretasikan dengan sangat hati-hati.

⚠ Definisi Kritis — Partial Regression Coefficient

Partial regression coefficient bⱼ mendeskripsikan dampak independent variable Xⱼ terhadap dependent variable Y, dengan menahan semua independent variable lain tetap konstan (holding all other independent variables constant).

Ini berbeda fundamental dari koefisien dalam simple regression. Kata kunci "ceteris paribus" (hal lain dianggap sama) ini adalah inti dari interpretasi koefisien multiple regression, dan sering menjadi jebakan di exam.

Misalnya, dalam persamaan:

Yᵢ = b₀ + b₁X₁ᵢ + b₂X₂ᵢ + b₃X₃ᵢ + εᵢ

Koefisien b₂ mengukur perubahan Y untuk setiap perubahan satu unit dalam X₂, dengan asumsi X₁ dan X₃ ditahan konstan. Koefisien estimasi ditulis dengan tanda ^ (hat): b̂₀, b̂₁, b̂₂, b̂₃.

2.3 Contoh Numerik Lengkap — Bond Index Returns

Mari kita lihat contoh konkret. Pertimbangkan persamaan estimated regression di mana monthly excess return dari sebuah bond index (RET) diregresi terhadap perubahan monthly government bond yields (BY) dan perubahan investment-grade credit spreads (CS). Regression ini menggunakan 60 observasi bulanan:

Model: Monthly Excess Return Bond Index vs. Bond Yields & Credit Spreads
RET = 0.0023 − 5.0585 × BY − 2.1901 × CS n = 60 monthly observations

Dari regression ini, kita dapat menyimpulkan empat hal:

  • Intercept (0.0023): Bond index menghasilkan return rata-rata 0.0023% per bulan, atau sekitar 0.028% per tahun, apabila perubahan government bond yields DAN perubahan credit spreads keduanya bernilai nol. Ini adalah baseline return ketika kedua faktor tidak bergerak.
  • Koefisien BY (−5.0585): Untuk setiap perubahan satu unit dalam monthly government bond yield (BY), return bond index berubah sebesar −5.0585%, dengan asumsi credit spread (CS) tidak berubah. Ini berarti bond index ini memiliki empirical duration sebesar 5.0585 — artinya durasi yang diestimasi dari data pasar, bukan dari perhitungan teoretis.
  • Koefisien CS (−2.1901): Kalau investment-grade credit spreads naik satu unit, return bond index berubah sebesar −2.1901%, dengan asumsi BY tidak berubah. Ini mencerminkan sensitivitas bond index terhadap perubahan credit conditions.
  • Prediksi: Untuk bulan di mana perubahan credit spreads = 0.001 dan perubahan government bond yields = 0.005, expected excess return bond index adalah:
    RET = 0.0023 − 5.0585(0.005) − 2.1901(0.001) = 0.0023 − 0.0253 − 0.0022 = −0.0252, atau −2.52%
2.4 Contoh Fama-French Three-Factor Model

Berikut adalah contoh lain yang sangat relevan dengan pekerjaan lo sebagai fund manager — regression monthly excess return suatu portfolio terhadap tiga Fama-French factors:

Fama-French Three-Factor Regression
RETRF = 1.5324 + 0.5892×MKTRF + (−0.8719)×SMB + (−0.0560)×HML

RETRF = portfolio excess return | MKTRF = market excess return | SMB = small-minus-big (size factor) | HML = high-minus-low (value factor) | Semua return dalam persentase penuh (1 untuk 1%)

Knowledge Check 2.1 — Interpretasi Koefisien Fama-French

(1) Interpretasikan intercept.

Jawaban: Apabila market excess return, SMB, dan HML masing-masing bernilai nol, kita mengekspektasikan return portfolio sebesar 1.5324%. Ini adalah alpha — return yang tidak dapat dijelaskan oleh ketiga faktor.

(2) Interpretasikan setiap slope coefficient.

Setiap koefisien diinterpretasikan dengan asumsi variabel lain ditahan konstan:
  • MKTRF (0.5892): Jika market return naik 1%, kita mengekspektasikan return portfolio naik 0.5892%. Portfolio ini memiliki beta pasar yang lebih rendah dari 1 — kurang sensitif terhadap pergerakan pasar secara keseluruhan.
  • SMB (−0.8719): Jika size factor returns naik 1% (artinya saham-saham kecil outperform saham-saham besar), kita mengekspektasikan return portfolio turun 0.8719%. Ini mengindikasikan portfolio ini memiliki bias ke saham-saham besar (large-cap tilt).
  • HML (−0.0560): Jika value factor returns naik 1% (artinya saham-saham value outperform saham-saham growth), kita mengekspektasikan return portfolio turun 0.056%. Koefisien ini sangat kecil dan kemungkinan tidak signifikan secara statistik — portfolio ini hampir tidak memiliki bias value/growth.

(3) Hitung predicted return portfolio jika MKTRF = 1, SMB = 4, dan HML = −2.

Kalkulasi:
RETRF = 1.5324 + 0.5892(1) − 0.8719(4) − 0.0560(−2) = 1.5324 + 0.5892 − 3.4876 + 0.1120 = −1.2540 ≈ −1.25%
Portfolio diperkirakan mengalami excess return negatif sebesar −1.25% dalam skenario ini, terutama karena faktor SMB yang sangat besar (4%) yang berdampak besar mengingat koefisien −0.8719.
3
Lima Asumsi Underlying OLS

Sebelum kita bisa melakukan statistical inference yang valid pada model multiple linear regression yang diestimasi menggunakan Ordinary Least Squares (OLS), kita perlu memastikan bahwa asumsi-asumsi yang mendasari model tersebut terpenuhi.

Dalam simple regression, ada empat asumsi yang perlu dipenuhi. Dalam multiple regression, asumsi-asumsi tersebut sedikit dimodifikasi untuk mencerminkan adanya beberapa independent variables:

  • 1
    Linearitas (Linearity)
    Hubungan antara dependent variable dan independent variables harus bersifat linear. Ini berarti model yang kita spesifikasikan (Yi = b₀ + b₁X₁ᵢ + ... + bₖXₖᵢ + εᵢ) adalah representasi yang tepat dari hubungan sebenarnya di antara variabel-variabel tersebut. Kalau hubungan sebenarnya bersifat non-linear (misalnya kuadratik atau eksponensial), menggunakan model linear akan menghasilkan koefisien yang bias.
    ⚠ Pelanggaran: Pola kurva/non-random pada residual vs. fitted plot atau residual vs. X plot
  • 2
    Homoskedastisitas (Homoskedasticity)
    Variance dari regression residuals harus sama untuk semua observasi — Var(εᵢ) = σ² untuk semua i. Kalau variance residual tidak konstan (heteroskedasticity), OLS masih menghasilkan estimasi yang unbiased, tetapi standard errors menjadi tidak valid, sehingga t-tests dan F-tests tidak bisa dipercaya.
    ⚠ Pelanggaran: Fan shape (kipas) atau pola melebar/menyempit pada residual plot
  • 3
    Independensi Error (Independence of Errors)
    Observasi-observasi harus bersifat independen satu sama lain. Implikasinya adalah regression residuals tidak berkorelasi antar observasi — Cov(εᵢ, εⱼ) = 0 untuk i ≠ j. Pelanggaran asumsi ini paling umum terjadi pada data time series, di mana residual periode sekarang berkorelasi dengan residual periode sebelumnya (serial correlation atau autocorrelation).
    ⚠ Pelanggaran: Pola sinusoidal/cyclical pada residual vs. time plot; Durbin-Watson statistic jauh dari 2
  • 4
    Normalitas (Normality)
    Regression residuals harus berdistribusi normal. Asumsi ini diperlukan agar t-tests dan F-tests menghasilkan p-values yang valid, terutama untuk sample size yang kecil. Untuk sample size yang besar, Central Limit Theorem memungkinkan inferensi yang valid meskipun residuals tidak perfectly normal.
    ⚠ Pelanggaran: Titik-titik menyimpang dari diagonal pada Q-Q plot, terutama di kedua ujung (fat tails)
  • 5
    Independensi Independent Variables (Independence of Independent Variables)
    Asumsi ini terdiri dari dua bagian:

    5a. Independent variables tidak bersifat random (non-stochastic).
    5b. Tidak ada hubungan linear yang eksak antara dua atau lebih independent variables, atau kombinasinya (no perfect multicollinearity).

    Asumsi independensi ini diperlukan agar koefisien bisa diestimasi. Kalau ada hubungan linear yang eksak antara independent variables, model tidak bisa diestimasi sama sekali. Dalam kasus yang lebih umum — di mana ada hubungan linear yang approxiate (multicollinearity) — model masih bisa diestimasi, namun interpretasinya menjadi bermasalah: koefisien mungkin memiliki tanda yang salah, confidence intervals yang sangat lebar, dan signifikansi statistik yang tidak dapat diandalkan.
    ⚠ Pelanggaran: Korelasi pairwise yang tinggi antar X; VIF (Variance Inflation Factor) yang besar
✅ Properti BLUE — Ketika Semua Asumsi Terpenuhi

Apabila semua lima asumsi terpenuhi, estimator OLS memiliki properti yang disebut BLUE — Best Linear Unbiased Estimator. Ini berarti:

Best: Di antara semua linear estimators yang tidak bias, OLS memiliki variance terkecil (Gauss-Markov Theorem).

Linear: Estimator adalah fungsi linear dari observed values.

Unbiased: Rata-rata dari estimasi koefisien, di atas semua possible samples, sama dengan nilai parameter sebenarnya.

⚠ Mnemonic: LINE + I

Linearity · Independence of errors · Normality · Equal variance (Homoskedasticity) · Independence of X's

Kalau semua terpenuhi → OLS adalah BLUE. Pelanggaran masing-masing asumsi menghasilkan masalah berbeda yang akan dibahas lebih mendalam di LM3 (Model Misspecification).

4
Diagnostic Plots — Cara Membaca & Menginterpretasikan

Regression software menghasilkan diagnostic plots yang merupakan alat yang sangat berguna untuk mendeteksi potensi pelanggaran asumsi-asumsi multiple linear regression. Untuk mengilustrasikan penggunaan plot-plot ini, kita akan menganalisis 10 tahun data monthly excess returns dari saham ABC menggunakan Fama-French three-factor model:

ABC_RETRFₜ = b₀ + b₁×MKTRFₜ + b₂×SMBₜ + b₃×HMLₜ + εₜ
4.1 Scatterplot Matrix (Pairs Plot)

Langkah pertama dalam analisis residual adalah melihat scatterplot matrix — matriks yang menampilkan scatterplot pairwise antara semua variabel dalam model. Dari matrix ini, kita bisa melihat:

  • Baris bawah (scatterplots antara ABC_RETRF dan setiap faktor):
    • Ada hubungan positif antara ABC_RETRF dan market factor (MKTRF) — masuk akal karena hampir semua saham bergerak searah dengan pasar.
    • Tampaknya tidak ada hubungan yang jelas antara ABC_RETRF dan size factor (SMB) di scatterplot. Namun ini bukan berarti SMB tidak signifikan dalam regression! Scatterplot hanya menampilkan hubungan bivariate, bukan hubungan parsial (partial) yang dikontrol oleh variabel lain. SMB bisa saja signifikan di regression ketika dikontrol MKTRF dan HML, meskipun bivariate plot-nya tidak menunjukkan hubungan yang jelas.
    • Ada hubungan negatif antara ABC_RETRF dan value factor (HML).
  • Scatterplot antar independent variables: SMB dan HML tampak memiliki sedikit atau tidak ada korelasi (garis regression-nya hampir datar). Ini adalah karakteristik yang diinginkan antara variabel-variabel penjelas — kalau ada korelasi tinggi di sini, kita perlu waspada terhadap multicollinearity.
💡 Perbedaan Bivariate Correlation vs. Partial Correlation

Ini adalah salah satu konsep yang sering membingungkan. Scatterplot matrix menampilkan hubungan bivariate — hubungan antara dua variabel secara terpisah, tanpa mengontrol variabel lain. Regression mengukur partial correlation — hubungan antara X dan Y setelah mengontrol semua X lainnya.

Inilah mengapa sebuah variabel bisa tampak tidak berkorelasi dengan Y di scatterplot, tetapi tetap signifikan dalam regression — karena regression "menyaring" informasi yang dibagi dengan variabel lain dan mengidentifikasi kontribusi unik masing-masing variabel.

4.3 Tiga Diagnostic Plot Utama

Setelah mengestimasi model, kita perlu memeriksa apakah asumsi-asumsi OLS terpenuhi melalui diagnostic plots. Ada tiga plot utama yang harus dikuasai:

📊 Plot 1: Residuals vs. Fitted Value (Predicted Value)

Tujuan: Mendeteksi heteroskedasticity, non-linearity, dan outliers.

Cara baca: Plot residual (εᵢ) di sumbu Y terhadap predicted value (Ŷᵢ) di sumbu X.

Apa yang diharapkan (asumsi terpenuhi): Titik-titik tersebar secara random di sekitar garis horizontal εᵢ = 0, tanpa ada pola yang jelas. Variance residual tampak konstan di seluruh range fitted values.

Tanda-tanda masalah:

— Fan shape (pola kipas): Residuals menyebar lebih lebar seiring bertambahnya fitted value → heteroskedasticity

— Pola kurva/U-shape: Ada hubungan non-linear yang tidak tertangkap oleh model → model misspecification

— Titik-titik yang jauh terisolasi: Potential outliers yang perlu diinvestigasi

Hasil untuk ABC: Tidak ada directional relationship yang terlihat — line referensi berada dekat εᵢ = 0. Namun ada tiga potential outliers (bulan 7, 25, dan 95) yang ditandai dengan square markers.

📊 Plot 2: Residuals vs. Each Independent Variable (X)

Tujuan: Mendeteksi apakah ada hubungan antara residuals dengan masing-masing X — yang mengindikasikan model misspecification atau heteroskedasticity yang berkaitan dengan X tertentu.

Cara baca: Buat satu plot terpisah untuk setiap independent variable — residual (εᵢ) di sumbu Y, nilai X di sumbu X.

Apa yang diharapkan: Titik-titik tersebar random tanpa pola direktional (tidak ada kecenderungan naik atau turun seiring berubahnya nilai X).

Tanda-tanda masalah:

— Pola direktional (tren naik/turun): Residuals berkorelasi dengan X tertentu → model misspecification, mungkin perlu transformasi X

— Fan shape untuk X tertentu: Heteroskedasticity yang berkaitan dengan variabel tersebut

Hasil untuk ABC: Ketiga plot (residual vs. MKTRF, vs. SMB, vs. HML) tidak menunjukkan directional relationship — tidak ada bukti pelanggaran asumsi. Tiga outlier yang sama (bulan 7, 25, 95) muncul konsisten di semua plot.

📊 Plot 3: Normal Q-Q Plot

Tujuan: Memeriksa apakah regression residuals berdistribusi normal (Asumsi 4).

Cara baca: Q-Q plot (Quantile-Quantile plot) membandingkan distribusi empiris dari standardized residuals dengan distribusi normal teoritis. Jika residuals normal, titik-titik seharusnya berada sepanjang diagonal garis 45 derajat.

Cara interpretasi: Kalau 5% observasi yang berdistribusi normal seharusnya berada di bawah −1.65 standard deviations, maka observasi pada persentil ke-5 seharusnya muncul pada −1.65 standard deviations di Q-Q plot.

Tanda-tanda masalah:

— Penyimpangan di kedua ujung (tails): Distribusi residuals memiliki fat tails — lebih banyak extreme values daripada yang diharapkan dari distribusi normal. Ini sangat umum terjadi pada data finansial.

— Titik-titik membentuk kurva S: Distribusi skewed (condong ke satu arah)

Hasil untuk ABC: Secara keseluruhan, titik-titik berada sepanjang diagonal — residuals mendekati normal. Namun observasi 25 dan 95 berada jauh di bawah garis diagonal setelah −2 standard deviations, sementara observasi 7 berada sedikit di atas garis sekitar +2.5 standard deviations. Ini mengkonfirmasi tiga outlier yang sama.

💡 Konsistensi Outlier Antar Plot

Perhatikan bahwa tiga observasi yang sama — bulan 7, 25, dan 95 — muncul sebagai potential outliers di semua tiga jenis diagnostic plot: residual vs. fitted, residual vs. setiap X, dan Q-Q plot. Konsistensi ini mengonfirmasi bahwa ketiga observasi tersebut memang genuine outliers yang perlu diinvestigasi.

Langkah selanjutnya adalah memeriksa apa yang terjadi pada bulan-bulan tersebut — apakah ada shocks dari faktor-faktor yang tidak dimasukkan ke dalam model? Misalnya, earnings surprise yang besar, kejadian korporat luar biasa, atau gangguan pasar yang tidak tertangkap oleh ketiga faktor Fama-French.

Knowledge Check 4.1 — Diagnostic Plots

(1) Apa tujuan dari scatterplot antara residuals dengan predicted value of CRYPTO?

Jawaban: Plot ini berguna untuk memeriksa apakah ada clustering atau pola yang mengindikasikan residuals tidak homoskedastis (variance tidak konstan) — fan shape adalah tanda klasiknya. Plot ini juga berguna untuk mengidentifikasi potential outliers.

(2) Apa tujuan dari plot residuals vs. GOLD (salah satu independent variable)?

Jawaban: Plot ini berguna untuk memeriksa apakah ada extreme values dari independent variable yang mungkin mempengaruhi estimated regression parameters (influential points), dan apakah ada hubungan antara residuals dengan independent variable tersebut — yang mengindikasikan model misspecification.

(3) Pairwise scatterplot digunakan untuk mendeteksi apa?

Jawaban: A — ada hubungan linear antara dependent dan independent variables. Pairwise scatterplot berguna untuk memvisualisasikan hubungan antara variabel-variabel dalam model. Ini bukan untuk mendeteksi heteroskedasticity (itu residual vs. fitted plot) maupun normalitas residuals (itu Q-Q plot).

(4) Q-Q plot digunakan untuk mendeteksi apa?

Jawaban: C — apakah regression residuals berdistribusi normal. Q-Q plot membandingkan distribusi empiris residuals dengan distribusi normal teoritis. Kalau titik-titik berada sepanjang diagonal, residuals mendekati normal. Penyimpangan di tails mengindikasikan fat-tailed distribution — yang sangat umum pada data finansial.

Case Vignette
Arif Nugroho, CFA candidate, adalah analis ekuitas di PT Garuda Investama, Jakarta. Ia sedang membangun model untuk memprediksi return bulanan saham-saham di sektor perbankan IDX menggunakan data bulanan 5 tahun terakhir (60 observasi). Model yang ia estimasikan adalah:

BANK_RETᵢ = b₀ + b₁×IHSG_RETᵢ + b₂×ΔNPLᵢ + b₃×ΔNIMᵢ + εᵢ

Hasil estimasi OLS menghasilkan: b₀ = 0.004, b₁ = 1.15, b₂ = −3.20, b₃ = 2.80.

Ketika Arif menjalankan diagnostic plots, ia menemukan tiga temuan penting: (1) Pada residual vs. fitted value plot, variance residual tampak membesar seiring meningkatnya fitted value — membentuk pola seperti kipas; (2) Pada Q-Q plot, titik-titik di kedua ujung menyimpang cukup jauh dari diagonal, mengindikasikan distribusi dengan ekor yang lebih tebal dari distribusi normal; (3) Ketika ia cek scatterplot matrix, korelasi antara ΔNPL dan ΔNIM adalah −0.72, yang cukup besar.

Rekannya menyarankan agar Arif menggunakan logistic regression untuk model ini, berargumen bahwa model linear tidak tepat karena ada masalah dengan residuals.
Question 1 of 4
Berdasarkan residual vs. fitted value plot yang berbentuk kipas (fan shape), pelanggaran asumsi OLS yang paling mungkin terjadi adalah:
B adalah jawaban yang benar. Fan shape (pola kipas) pada residual vs. fitted plot adalah tanda klasik dari heteroskedasticity — Var(εᵢ) meningkat seiring meningkatnya fitted value, melanggar asumsi homoskedasticity (Equal variance). Serial correlation biasanya terdeteksi lewat pola cyclical/sinusoidal pada residual vs. time plot, bukan pada residual vs. fitted plot. Multicollinearity tidak tampak di residual plots — ia terdeteksi dari korelasi pairwise antar X atau VIF, dan dampaknya adalah pada koefisien (bukan pada pola residual).
Question 2 of 4
Koefisien b₂ = −3.20 pada ΔNPL (perubahan rasio NPL) diinterpretasikan sebagai:
A adalah jawaban yang benar. Ini adalah partial regression coefficient — interpretasinya selalu holding other independent variables constant (ceteris paribus). Inilah yang membedakan multiple regression dari simple regression. Opsi B salah karena menghilangkan kata kunci "holding others constant" — tanpa frasa itu, interpretasinya menjadi interpretasi simple regression coefficient, bukan partial. Opsi C salah total — koefisien regression bukan korelasi; korelasi dibatasi antara −1 dan +1, sementara regression coefficient bisa bernilai berapa saja.
Question 3 of 4
Temuan Q-Q plot yang menunjukkan titik-titik menyimpang jauh dari diagonal di kedua ujung paling tepat mengindikasikan:
B adalah jawaban yang benar. Q-Q plot membandingkan distribusi empiris residuals dengan distribusi normal teoritis. Penyimpangan di kedua ujung (tails) — khususnya titik-titik yang berada di luar diagonal setelah ±2 standard deviations — mengindikasikan distribusi residuals memiliki fat tails (lebih banyak extreme values daripada yang diharapkan dari distribusi normal). Ini adalah pelanggaran asumsi normality dan sangat umum terjadi pada data time series finansial. Heteroskedasticity terdeteksi dari residual vs. fitted plot (bukan Q-Q plot). Omitted variable bias adalah masalah model specification yang tidak dapat dideteksi langsung dari Q-Q plot.
Question 4 of 4
Saran rekan Arif untuk menggunakan logistic regression sebagai respons terhadap masalah pada residuals adalah:
C adalah jawaban yang benar. Pemilihan antara OLS dan logistic regression didasarkan pada sifat dependent variable, bukan pada kondisi residuals. Logistic regression digunakan ketika dependent variable adalah diskrit/biner — misalnya "apakah bank ini akan default dalam 12 bulan ke depan?" (1=ya, 0=tidak). BANK_RET adalah return persentase bulanan — ini adalah variabel kontinu, sehingga OLS adalah pendekatan yang tepat. Masalah heteroskedasticity diselesaikan dengan Weighted Least Squares (WLS) atau robust standard errors (Huber-White), dan masalah non-normality pada large samples biasanya tidak terlalu mengkhawatirkan karena Central Limit Theorem.
