Theory · 30 min
1
ANOVA Table — Anatomy & Interpretation

ANOVA table memecah total variasi Y (SST) menjadi dua komponen:

SourcedfSum of SquaresMean SquareF-stat
Regression k SSR = Σ(Ŷᵢ − Ȳ)² MSR = SSR/k MSR/MSE
Residual (Error) n−k−1 SSE = Σ(Yᵢ − Ŷᵢ)² MSE = SSE/(n−k−1)
Total n−1 SST = Σ(Yᵢ − Ȳ)² — —

R² dari ANOVA:

R² = SSR / SST // Proportion of Y variation explained by the model

F-statistic (General Linear F-test):
H₀: b₁ = b₂ = ... = bₖ = 0 (tidak ada X yang signifikan)
Hₐ: at least one bⱼ ≠ 0

F = MSR / MSE ~ F(k, n−k−1) // Reject H₀ if F_calculated > F_critical
Contoh baca ANOVA (dari CFA): 5-factor model, n=50:
SSR=90.62, SSE=56.62, SST=147.24, df_regression=5, df_residual=44
→ R² = 90.62/147.24 = 0.6155 → F = (90.62/5)/(56.62/44) = 18.12/1.29 = 14.09 → Reject H₀ (at least one factor significant)
2
R² vs Adjusted R² — Kenapa Adjusted Lebih Baik

Masalah dengan R²: R² tidak pernah turun ketika variabel baru ditambahkan — bahkan variabel yang tidak relevan sekalipun akan meningkatkan (atau minimal maintain) R². Ini mendorong overfitting.

Adjusted R² = 1 − [(n−1)/(n−k−1)] × (1 − R²) // k = jumlah independent variables

Adjusted R² memberikan penalti untuk setiap variabel yang ditambahkan. Hasilnya:

Kondisi Saat Tambah Variabel BaruEfek pada Adj. R²
|t-statistic variabel baru| > 1.0 ↑ Naik
|t-statistic variabel baru| < 1.0 ↓ Turun
⚠ Exam Trap

Bar |t| > 1.0 untuk meningkatkan Adjusted R² itu rendah — jauh di bawah threshold signifikansi 5% (yang butuh t ≈ 2.0). Artinya Adjusted R² bisa naik bahkan untuk variabel yang secara statistik tidak signifikan. Jangan gunakan Adjusted R² sebagai satu-satunya ukuran model quality.

Catatan penting: Adjusted R² bisa negatif (kalau R² sangat kecil dan k besar). Ini tidak punya interpretasi intuitif seperti "persentase variasi yang dijelaskan" — tidak seperti R² dalam simple regression.

3
AIC & BIC — Model Selection Criteria

Ketika membandingkan beberapa model dengan dependent variable yang sama, gunakan AIC atau BIC. Keduanya mengukur parsimony — model yang lebih sederhana lebih disukai kalau fit-nya serupa.

AIC = n × ln(SSE/n) + 2(k+1) BIC = n × ln(SSE/n) + ln(n)(k+1) // Lower is better untuk keduanya // Penalty AIC: 2(k+1) | Penalty BIC: ln(n)(k+1)
KriteriaPenaltyLebih Ketat?Gunakan Kalau...
AIC 2(k+1) Lebih longgar Tujuan model adalah prediksi
BIC ln(n)(k+1) Lebih ketat Tujuan model adalah goodness of fit
Intuisi: BIC lebih ketat karena ln(n) > 2 untuk semua n ≥ 8. Artinya BIC mengenakan penalti lebih besar pada model yang kompleks, sehingga cenderung memilih model yang lebih parsimonious (sedikit variabel).
⚠ Exam Trap

Nilai absolut AIC atau BIC sendiri tidak bermakna — yang penting adalah perbandingan relatif antar model. AIC model A = 18 vs model B = 22 → model A lebih baik. Tapi AIC = 18 sendiri tidak bisa diinterpretasikan apa pun.

4
Joint F-Test — Restricted vs Unrestricted Model

Kadang kita ingin test apakah sekelompok variabel secara bersama-sama signifikan, bukan satu per satu. Gunakan joint F-test untuk nested models:

Setup:
Unrestricted model: Y = b₀ + b₁X₁ + b₂X₂ + b₃X₃ + b₄X₄ + b₅X₅ + ε
Restricted model: Y = b₀ + b₁X₁ + b₂X₂ + b₃X₃ + ε
H₀: b₄ = b₅ = 0  |  Hₐ: at least one ≠ 0

F = [(SSE_restricted − SSE_unrestricted) / q] / [SSE_unrestricted / (n−k−1)] // q = jumlah variabel yang di-restrict (di-drop) // Degrees of freedom: q (numerator), n−k−1 (denominator)
Intuisi: Kalau F besar → dropping X₄ dan X₅ menyebabkan SSE naik signifikan → mereka memang berkontribusi → tolak H₀ → keep mereka. Kalau F kecil → SSE hampir tidak berubah → tidak ada kontribusi signifikan → fail to reject H₀ → parsimonious model lebih baik.
💡 Contoh Numerik (dari CFA)

SSE_restricted (3 factors) = 64.52, SSE_unrestricted (5 factors) = 56.62, q=2, n−k−1=44
F = [(64.52 − 56.62)/2] / [56.62/44] = 3.95/1.29 = 3.07
Critical F (2, 44) at 1% = 5.12 → Fail to reject H₀ → factors 4 & 5 tidak signifikan bersama-sama → gunakan restricted 3-factor model.

⚠ Exam Trap

Dua variabel bisa signifikan secara individual (t-test) tapi tidak signifikan secara joint (F-test), atau sebaliknya — terutama kalau ada korelasi antar X. Selalu lakukan kedua test kalau relevan. Jangan assume individual significance = joint significance.

Quick Reference Cheat Sheet

Adjusted R²
↑ if |t_new| > 1.0 | ↓ if |t_new| < 1.0
General F-test
F = MSR/MSE | df: k, n−k−1
Joint F-test
F = [(SSE_R−SSE_U)/q] / [SSE_U/(n−k−1)]

Practice — Item Set Vignette
Format CFA L2 · 4 soal · ~30 menit · Jawab dulu sebelum lihat penjelasan
Case Vignette
Siti Rahayu, CFA, adalah kepala riset kuantitatif di PT Mandiri Investasi, Jakarta. Timnya sedang membangun model untuk memprediksi return bulanan saham-saham sektor konsumer di IDX. Mereka mengestimasi tiga model dengan dependent variable yang sama (CONS_RET = monthly return saham konsumer) menggunakan 60 observasi bulanan:

Model A: CONS_RET = b₀ + b₁×IHSG_RET + ε
Model B: CONS_RET = b₀ + b₁×IHSG_RET + b₂×ΔCPI + b₃×ΔUSD_IDR + ε
Model C: CONS_RET = b₀ + b₁×IHSG_RET + b₂×ΔCPI + b₃×ΔUSD_IDR + b₄×ΔOIL + b₅×ΔINT_RATE + ε

Hasil goodness-of-fit:
ModelkSSER²Adj. R²AICBIC
A142.800.6120.60618.421.2
B338.100.6540.63616.822.4
C537.600.6590.62818.226.9

Untuk Model C, t-statistic untuk ΔOIL = 0.62 dan ΔINT_RATE = −0.85. Siti ingin menguji apakah ΔOIL dan ΔINT_RATE secara bersama-sama berkontribusi signifikan pada Model C, menggunakan α = 5%. Critical F(2, 54) at 5% = 3.17.
Question 1 of 4
Dari perbandingan Model A dan Model C, manakah pernyataan yang paling tepat mengenai R² dan Adjusted R²?
B adalah jawaban yang benar. R² tidak pernah turun saat variabel baru ditambahkan — itulah mengapa R² Model C (0.659) > Model A (0.612). Namun Adjusted R² menggunakan penalti: kalau |t-statistic| variabel baru < 1.0, Adjusted R² turun. Di sini, |t| ΔOIL = 0.62 < 1.0 dan |t| ΔINT_RATE = 0.85 < 1.0 → Adjusted R² Model C (0.628) lebih rendah dari Model B (0.636). Opsi C salah karena R² tidak bisa turun saat variabel ditambahkan.
Question 2 of 4
Jika Siti bertujuan membangun model untuk prediksi return saham konsumer ke depan, model mana yang paling sesuai dipilih berdasarkan kriteria informasi?
B adalah jawaban yang benar. Untuk tujuan prediksi, gunakan AIC — pilih model dengan AIC terendah. Model B memiliki AIC = 16.8 (terendah di antara ketiganya). Opsi A salah karena R² tidak cocok untuk pemilihan model karena selalu naik. Opsi C salah karena BIC digunakan untuk tujuan goodness of fit, bukan prediksi — dan meski Model A punya BIC rendah, AIC-nya bukan yang terendah.
Question 3 of 4
Untuk joint F-test apakah ΔOIL dan ΔINT_RATE secara bersama-sama signifikan (Model B = restricted, Model C = unrestricted), berapakah nilai F-statistic yang dihitung?
C adalah jawaban yang benar. Untuk joint F-test Model B (restricted, 3 vars) vs Model C (unrestricted, 5 vars), q=2 variabel yang di-restrict (ΔOIL dan ΔINT_RATE), dan SSE yang digunakan adalah SSE Model B (restricted) = 38.10 dan SSE Model C (unrestricted) = 37.60:
F = [(38.10 − 37.60)/2] / [37.60/(60−5−1)] = (0.50/2) / (37.60/54) = 0.25/0.696 ≈ 0.36
F = 0.36 < F_critical = 3.17 → Fail to reject H₀ → ΔOIL dan ΔINT_RATE tidak signifikan bersama-sama. Opsi A salah karena menggunakan SSE Model A (bukan Model B) sebagai restricted. Opsi B salah karena menggunakan SSE yang keliru.
Question 4 of 4
Berdasarkan semua analisis (Adjusted R², AIC, BIC, joint F-test), kesimpulan yang paling tepat adalah:
B adalah jawaban yang benar. Semua bukti mengarah ke Model B: (1) Adjusted R² Model B (0.636) adalah tertinggi, (2) AIC Model B (16.8) adalah terendah untuk tujuan prediksi, (3) Joint F-test menunjukkan ΔOIL dan ΔINT_RATE tidak signifikan bersama-sama → tidak menambah nilai. BIC Model A memang lebih rendah dari Model B, tapi Siti menyatakan tujuannya adalah prediksi → AIC lebih relevan, dan Model B menang di AIC. Ini adalah contoh klasik di mana berbagai goodness-of-fit metrics mengarah ke kesimpulan konsisten.
