1
The Multiple Regression Equation

Model multiple regression menjelaskan variasi dependent variable Y menggunakan dua atau lebih independent variables:

Yi = b0 + b1X1i + b2X2i + ... + bkXki + εi
// i = 1, 2, ..., n observations; k = jumlah independent variables

b₀ (intercept): expected value of Y kalau semua X = 0.

b₁...b_k (slope / partial regression coefficients): perubahan Y untuk satu unit perubahan X_j, holding semua X lain constant. Ini yang bikin "partial" — beda dari simple regression.

Contoh konkret (dari CFA): RET = 0.0023 − 5.0585×BY − 2.1901×CS
Koefisien BY = −5.0585 artinya: kalau government bond yield naik 1 unit, bond index return turun 5.0585%, dengan asumsi credit spread (CS) tidak berubah. Ini adalah empirical duration.

Model diestimasi dengan Ordinary Least Squares (OLS) — meminimumkan jumlah kuadrat residual (SSR = Σε²).

2
5 Asumsi OLS — "LINE + I"
#AsumsiApa yang DilanggarStatus
1 Linearity
Hubungan Y dan X harus linear		 Non-linear pattern di residual plot L
2 Independence of Errors
Residual tidak berkorelasi antar observasi		 Serial correlation → SE bias I
3 Normality
Residual berdistribusi normal		 Fat tails → inference tidak valid N
4 Equal variance (Homoskedasticity)
Var(ε) sama untuk semua observasi		 Heteroskedasticity → SE bias E
5 Independence of X's
Tidak ada exact linear relation antar X		 Multicollinearity → koefisien tidak reliable +I
Mnemonic: LINE + Independence of X. Kalau semua asumsi terpenuhi → OLS adalah BLUE (Best Linear Unbiased Estimator).
3
Diagnostic Plots — Cara Baca

Tiga plot utama yang harus lo bisa interpretasikan:

① Residual vs Fitted Value
Kalau asumsi terpenuhi: residual tersebar random (no pattern) di sekitar 0. Kalau ada fan shape → heteroskedasticity. Kalau ada curve → non-linearity.

② Residual vs Each X
Sama seperti di atas tapi per variabel. Tidak boleh ada pola arah (directional relationship). Outlier yang muncul konsisten di semua plot = likely genuine outlier.

③ Normal Q-Q Plot
Bandingkan distribusi residual dengan distribusi normal teoritis. Kalau residual normal → titik-titik seharusnya jatuh sepanjang diagonal. Penyimpangan di ujung (tails) = fat tails / outliers.

⚠ Exam Trap

Jangan bingung antara pairwise correlation antar X vs partial correlation dalam regression. Di scatterplot matrix, SMB dan ABC_RETRF bisa tampak tidak berkorelasi — tapi SMB tetap bisa signifikan dalam multiple regression karena regression picks up partial effect setelah mengontrol variabel lain.

Outliers: Titik yang muncul sebagai outlier di semua diagnostic plots (residual vs fitted, residual vs X, Q-Q plot) adalah kandidat kuat untuk investigasi — cek apakah ada event khusus di periode tersebut.

4
Use Cases Multiple Regression di Investasi

CFA sering tanya: "kapan pakai multiple regression?" Tiga use case utama:

① Identify relationships — Fama-French 5-factor model untuk decompose return driver suatu portfolio (market, size, value, profitability, investment)

② Test theories — Apakah financial leverage, profitability, revenue growth bisa memprediksi financial distress?

③ Forecast — Estimasi expected return berdasarkan macro factors

Relevansi untuk lo sebagai fund manager: Setiap kali lo attributing portfolio returns, lo essentially doing multiple regression — decomposing outperformance/underperformance ke faktor-faktor spesifik. Fama-French yang lo pakai itu multiple regression applied.

Practice — Item Set Vignette
Format CFA L2 · 4 soal · ~30 menit · Jawab dulu sebelum lihat penjelasan
Case Vignette
Arif Nugroho, CFA candidate, adalah analis ekuitas di PT Garuda Investama, Jakarta. Ia sedang membangun model untuk memprediksi return saham-saham di sektor perbankan IDX menggunakan data bulanan 5 tahun terakhir (60 observasi). Model yang ia estimasikan adalah:

BANK_RETi = b₀ + b₁×IHSG_RETi + b₂×ΔNPLi + b₃×ΔNIMi + εi

Hasil estimasi OLS: b₀ = 0.004, b₁ = 1.15, b₂ = −3.20, b₃ = 2.80.

Ketika Arif menjalankan diagnostic plots, ia menemukan: (1) pada residual vs fitted value plot, variance residual tampak membesar seiring meningkatnya fitted value — membentuk pola seperti kipas; (2) pada Q-Q plot, titik-titik di kedua ujung menyimpang cukup jauh dari diagonal; (3) ketika ia cek scatterplot matrix, korelasi antara ΔNPL dan ΔNIM adalah −0.72. Rekannya menyarankan agar Arif menggunakan logistic regression untuk model ini.
Question 1 of 4
Berdasarkan residual vs fitted value plot yang berbentuk kipas (fan shape), pelanggaran asumsi OLS yang paling mungkin terjadi adalah:
B adalah jawaban yang benar. Fan shape pada residual vs fitted plot adalah signature klasik heteroskedasticity — variance residual (Var(ε)) meningkat seiring bertambahnya fitted value, melanggar asumsi homoskedasticity (Equal variance). Serial correlation biasanya terdeteksi lewat pola cyclical pada residual vs time plot. Multicollinearity tidak tampak di residual plot tapi dari korelasi antar X.
Question 2 of 4
Koefisien b₂ = −3.20 pada ΔNPL (perubahan rasio NPL) diinterpretasikan sebagai:
A adalah jawaban yang benar. Ini adalah partial regression coefficient — interpretasinya selalu holding other independent variables constant. Ini yang membedakan multiple regression dari simple regression. Opsi B salah karena mengabaikan kata "partial" — tanpa holding others constant, itu adalah simple regression coefficient, bukan partial. Opsi C salah karena koefisien bukan korelasi.
Question 3 of 4
Temuan Q-Q plot yang menunjukkan titik-titik menyimpang di kedua ujung diagonal mengindikasikan:
B adalah jawaban yang benar. Q-Q plot membandingkan distribusi empiris residual dengan distribusi normal teoritis. Penyimpangan di kedua ujung (tails) mengindikasikan distribusi residual punya lebih banyak extreme values dari yang diharapkan normal — ini disebut fat tails, yang melanggar asumsi normality. Heteroskedasticity terdeteksi di residual vs fitted plot (bukan Q-Q). Omitted variable tidak bisa dideteksi dari Q-Q plot.
Question 4 of 4
Saran rekan Arif untuk menggunakan logistic regression adalah:
C adalah jawaban yang benar. Logistic regression digunakan ketika dependent variable adalah diskrit — misalnya "apakah saham akan outperform atau tidak?" (1/0). BANK_RET adalah continuous variable (return dalam %), sehingga OLS (dengan koreksi untuk heteroskedasticity jika perlu) adalah pilihan yang tepat. Permasalahan heteroskedasticity diselesaikan dengan WLS atau robust standard errors, bukan dengan mengganti ke logistic regression.
