1
Definisi & Konstruksi Dummy Variable

Dalam praktik analisis investasi, kita sering perlu memasukkan informasi kualitatif ke dalam model regression. Misalnya: apakah perusahaan termasuk sektor tertentu? Apakah suatu negara adalah emerging market atau developed market? Apakah suatu periode berada sebelum atau sesudah perubahan regulasi? Informasi seperti ini tidak bisa direpresentasikan sebagai variabel kontinu biasa — di sinilah peran dummy variable (atau indicator variable).

Dummy variable adalah variabel independen yang hanya mengambil dua nilai: 1 jika suatu kondisi terpenuhi (true) dan 0 jika tidak terpenuhi (false). Dengan dummy variable, kita bisa membedakan "grup" atau "kategori" data di dalam satu model regression yang sama.

1.1 Sumber Dummy Variable

Dummy variable dapat berasal dari beberapa sumber:

  • Properti inheren data — misalnya keanggotaan industri (Technology, Financial Services, dsb.). Ini adalah karakteristik alami yang melekat pada observasi.
  • Kondisi true/false dari data — misalnya tanggal sebelum atau sesudah suatu event pasar penting (krisis, perubahan kebijakan). Di sini kita menandai observasi berdasarkan timing.
  • Konstruksi dari karakteristik data — misalnya apakah penjualan perusahaan di atas atau di bawah suatu threshold. Kita mengubah variabel kontinu menjadi kategori biner.
1.2 Aturan n − 1: Menghindari Dummy Variable Trap

Aturan paling fundamental dalam menggunakan dummy variables: jika kita ingin membedakan n kategori, kita membutuhkan tepat n − 1 dummy variables. Kategori yang tidak mendapat dummy menjadi base group (atau control group), dan koefisien setiap dummy diinterpretasikan relatif terhadap base group tersebut.

⚠ Dummy Variable Trap

Jika kita menggunakan n dummy variables untuk n kategori (bukan n − 1), maka seluruh dummy akan berjumlah sama persis dengan variabel intercept — ini menciptakan perfect multicollinearity. Software regression akan gagal atau menghasilkan output yang tidak bermakna. Ini disebut dummy variable trap dan merupakan pelanggaran asumsi "no exact linear relationship among independent variables".

Knowledge Check 1

Seorang analis ingin memasukkan variabel klasifikasi negara (Emerging Market, Frontier Market, Developed Market) ke dalam model regression return ekuitas. Berapa dummy variable yang dibutuhkan, dan bagaimana interpretasinya?

Dibutuhkan 2 dummy variables (misalnya D_EM dan D_Frontier, dengan Developed Market sebagai base group). Koefisien D_EM menunjukkan perbedaan rata-rata return emerging markets relatif terhadap developed markets. Koefisien D_Frontier menunjukkan perbedaan rata-rata return frontier markets relatif terhadap developed markets. Kedua koefisien diinterpretasikan holding other variables constant.
2
Intercept Dummy — Pergeseran Level

Jenis dummy variable yang paling umum adalah intercept dummy. Sesuai namanya, dummy ini mengubah intercept (level) dari garis regresi — bukan slope-nya. Artinya, dua kelompok data memiliki kemiringan yang sama terhadap variabel independen X, tetapi "titik awal" (level) mereka berbeda.

2.1 Model dengan Intercept Dummy

Pertimbangkan model sederhana dengan satu variabel kontinu X dan satu intercept dummy D:

Model Intercept Dummy Yi = b₀ + d₀D + b₁Xi + εi d₀ = koefisien intercept dummy

Model tunggal ini sebenarnya mengestimasi dua garis regresi yang berbeda tergantung nilai D:

  • Jika D = 0 (base group): Y = b₀ + b₁X + ε → intercept = b₀
  • Jika D = 1 (treatment group): Y = (b₀ + d₀) + b₁X + ε → intercept = b₀ + d₀

Perhatikan bahwa slope b₁ tidak berubah — kedua garis berjalan paralel. Perbedaan antara keduanya hanyalah "jarak vertikal" sebesar d₀. Nilai d₀ bisa positif (garis D = 1 di atas base) atau negatif (garis D = 1 di bawah base).

3
Slope Dummy & Interaction Terms

Intercept dummy mengasumsikan bahwa hubungan antara Y dan X (slope) sama untuk semua kategori — yang berbeda hanyalah level. Tapi bagaimana kalau kita curiga bahwa slope-nya juga berbeda? Misalnya, pengaruh usia dana terhadap return mungkin lebih kuat untuk growth funds dibanding value funds. Di sinilah slope dummy berperan.

3.1 Model dengan Slope Dummy
Model Slope Dummy Yi = b₀ + b₁Xi + d₁(Di × Xi) + εi D × X = interaction term (perkalian dummy dengan variabel kontinu)

Slope dummy menciptakan interaction term — perkalian antara dummy variable D dan variabel kontinu X. Ini memungkinkan slope berubah tergantung kategori:

  • Jika D = 0 (base): Y = b₀ + b₁X + ε → slope = b₁
  • Jika D = 1: Y = b₀ + (b₁ + d₁)X + ε → slope = b₁ + d₁

Berbeda dengan intercept dummy, kedua garis sekarang memiliki kemiringan berbeda tetapi berangkat dari intercept yang sama (b₀). Perbedaan slope adalah d₁ — bisa positif (slope lebih curam untuk D = 1) atau negatif (slope lebih landai).

3.2 Model Kombinasi: Intercept + Slope Dummy

Dalam praktik, kita sering menggunakan keduanya — intercept dummy DAN slope dummy — dalam satu model:

Model Kombinasi Yi = b₀ + d₀D + b₁Xi + d₁(Di × Xi) + εi Memungkinkan perubahan intercept DAN slope sekaligus
  • Jika D = 0: Y = b₀ + b₁X → intercept = b₀, slope = b₁
  • Jika D = 1: Y = (b₀ + d₀) + (b₁ + d₁)X → intercept = b₀ + d₀, slope = b₁ + d₁

⚠ Kesalahan Umum di Ujian

Banyak kandidat yang keliru menginterpretasikan koefisien interaction term sebagai total slope untuk kategori D = 1. Ingat: total slope = original slope + interaction coefficient. Koefisien interaction hanyalah selisih slope antara treatment group dan base group.

Knowledge Check 3

Model: RET = 0.50 − 0.5REG + 0.4MKTSH − 0.2(REG × MKTSH), di mana REG = 1 jika perusahaan regulated. Untuk perusahaan regulated dengan MKTSH = 30%, berapakah expected return?

RET = 0.50 − 0.5(1) + 0.4(0.30) − 0.2(1)(0.30)
= 0.50 − 0.50 + 0.12 − 0.06 = 0.06 atau 6%.
Untuk non-regulated (REG = 0) dengan MKTSH yang sama: RET = 0.50 + 0.4(0.30) = 0.62 atau 62%. Perbedaan signifikan ini menunjukkan regulated firms memiliki return yang jauh lebih rendah, dan efek market share juga lebih lemah (slope 0.2 vs 0.4).

Berapa slope market share untuk regulated vs non-regulated firms?

Non-regulated: slope MKTSH = 0.4 (karena REG = 0, interaction term hilang).
Regulated: slope MKTSH = 0.4 + (−0.2) = 0.2.
Ini artinya setiap peningkatan market share 1%, return regulated firm hanya naik 0.2% sedangkan non-regulated naik 0.4%. Regulated firms mendapat manfaat yang lebih kecil dari peningkatan market share.
4
Structural Breaks & Seasonal Dummies

Salah satu aplikasi paling penting dari dummy variables adalah mendeteksi structural breaks — perubahan mendadak dalam hubungan antar variabel yang disebabkan oleh event besar seperti perubahan regulasi, krisis finansial, atau pergeseran rezim kebijakan. Dummy variables memungkinkan kita menguji secara formal apakah model regression "berubah" pada titik waktu tertentu.

4.1 Mendeteksi Structural Breaks dengan Dummies

Cara kerjanya sederhana: kita membuat dummy variable yang bernilai 1 untuk observasi setelah event yang dicurigai dan 0 untuk sebelumnya. Dummy ini bisa berfungsi sebagai intercept dummy (menguji perubahan level), slope dummy (menguji perubahan sensitivitas), atau keduanya.

Test Structural Break Yi = b₀ + d₀POST + b₁Xi + d₁(POST × Xi) + εi POST = 1 untuk periode setelah event, 0 sebelumnya d₀ menguji perubahan intercept; d₁ menguji perubahan slope

Jika d₀ signifikan → ada structural break pada level (intercept shift). Jika d₁ signifikan → ada structural break pada sensitivitas (slope shift). Jika keduanya signifikan → perubahan total pada hubungan Y-X.

💡 Konteks Indonesia

Bayangkan model return IHSG: RET = b₀ + b₁×GDP + b₂×RATE + d₀×POST_COVID + d₁×(POST_COVID × GDP). Jika d₀ signifikan negatif, artinya level return memang turun secara struktural pasca-COVID. Jika d₁ signifikan, artinya sensitivitas return terhadap pertumbuhan GDP juga berubah.

4.2 Seasonal Dummies

Dummy variables juga berguna untuk menangkap efek musiman (seasonality) dalam data. Jika kita menggunakan data kuartalan, kita membuat 3 dummy variables untuk 4 kuartal (sesuai aturan n − 1):

  • Q1 = 1 jika observasi di kuartal 1, 0 jika tidak
  • Q2 = 1 jika observasi di kuartal 2, 0 jika tidak
  • Q3 = 1 jika observasi di kuartal 3, 0 jika tidak
  • Q4 = base group (semua dummies = 0)

Koefisien Q1 menunjukkan seberapa besar rata-rata Y di kuartal 1 berbeda dari Q4 (base), holding other variables constant. Ini berguna untuk menangkap fenomena seperti "January effect" (return saham cenderung lebih tinggi di Januari) atau pola earnings musiman.

Untuk data bulanan, dibutuhkan 11 dummy variables (12 bulan − 1). Untuk data harian berdasarkan hari dalam seminggu, dibutuhkan 4 dummies (5 hari perdagangan − 1).

Knowledge Check 4

Seorang analis menggunakan data bulanan selama 10 tahun (120 observasi) untuk model return saham dan ingin menguji apakah ada efek musiman. Berapa total dummy variable yang diperlukan untuk menangkap efek bulan, dan berapa banyak degrees of freedom yang "hilang" dibanding model tanpa seasonal dummies?

Diperlukan 11 dummy variables (12 bulan − 1, satu bulan menjadi base). Setiap dummy yang ditambahkan mengurangi degrees of freedom sebanyak 1. Jadi total 11 degrees of freedom hilang dibanding model tanpa seasonal dummies. Ini trade-off yang perlu dipertimbangkan: seasonal dummies bisa meningkatkan goodness-of-fit, tapi mengurangi precision (degrees of freedom). Apakah trade-off ini worthwhile tergantung pada apakah seasonal effects benar-benar signifikan.
5
Uji Signifikansi & Interpretasi Komprehensif

Dummy variables diuji signifikansinya dengan cara yang sama seperti variabel kontinu lainnya — menggunakan individual t-tests. Null hypothesis: koefisien dummy = 0 (artinya tidak ada perbedaan antara treatment group dan base group). Jika t-statistic signifikan (p-value < α), kita menyimpulkan ada perbedaan yang signifikan secara statistik.

5.2 Memilih Tipe Dummy yang Tepat

Pertanyaan kunci yang harus dijawab seorang analis:

  • "Apakah saya hanya ingin mengetahui perbedaan level antar grup?" → Gunakan intercept dummy saja. Contoh: apakah rata-rata return emerging market lebih tinggi dari developed market?
  • "Apakah saya ingin mengetahui apakah sensitivitas Y terhadap X berbeda antar grup?" → Gunakan slope dummy (interaction term). Contoh: apakah sensitivitas return terhadap GDP growth berbeda untuk emerging vs developed?
  • "Apakah saya curiga bahwa SELURUH hubungan berbeda?" → Gunakan keduanya (intercept + slope dummies). Ini model yang paling fleksibel tetapi juga paling boros degrees of freedom.
5.3 Contoh Komprehensif: Regulated vs Non-Regulated Firms

Model: RET = 0.50 − 0.5REG + 0.4MKTSH − 0.2(REG × MKTSH), di mana REG = 1 jika regulated, MKTSH = market share.

Untuk non-regulated firms (REG = 0):

  • RET = 0.50 + 0.4MKTSH → intercept = 0.50, slope MKTSH = 0.4

Untuk regulated firms (REG = 1):

  • RET = (0.50 − 0.50) + (0.4 − 0.2)MKTSH = 0 + 0.2MKTSH → intercept = 0, slope MKTSH = 0.2

Insight: Regulated firms memiliki return yang lebih rendah (intercept turun 0.50) DAN manfaat yang lebih kecil dari peningkatan market share (slope turun dari 0.4 ke 0.2). Regulasi secara simultan menekan level return dan mengurangi efektivitas market share dalam mendongkrak profitabilitas.

✅ Tips Ujian CFA

Di ujian CFA Level II, pertanyaan tentang dummy variables biasanya meminta kamu untuk: (1) menulis model equation berdasarkan deskripsi verbal, (2) menginterpretasikan koefisien untuk tiap kategori, atau (3) menghitung predicted value untuk observasi tertentu. Pastikan selalu menjumlahkan original coefficient + dummy coefficient untuk mendapat total effect kategori D = 1.

Knowledge Check 5

Seorang MD di hedge fund ingin mempelajari hubungan antara stock return dan GDP growth untuk beberapa negara, menggunakan binary variable EM = 1 (emerging) dan 0 (developed). Ia mempertimbangkan tiga opsi modeling: (A) slope dummy saja, (B) intercept dummy saja, (C) intercept + interaction term. Yang mana paling tepat untuk mengklasifikasikan negara berdasarkan development status?

B — intercept dummy adalah yang paling tepat untuk tujuan ini. Intercept dummy memungkinkan klasifikasi negara berdasarkan status emerging vs developed dan menguji apakah ada perbedaan level return rata-rata antara kedua kelompok. Jika MD juga ingin menguji apakah sensitivitas return terhadap GDP growth berbeda, barulah ia perlu menambahkan interaction term (opsi C). Slope dummy saja (opsi A) tidak cukup karena hanya mengubah kemiringan tanpa membedakan level.

Case Vignette
Rina Permata, CFA, bekerja sebagai quantitative analyst di PT Nusantara Asset Management, Jakarta. Ia membangun model untuk menjelaskan return bulanan saham-saham sektor perbankan yang terdaftar di IDX menggunakan data 96 observasi (Januari 2018 – Desember 2025).

Model 1 (Intercept Dummy Only):
BANK_RETi = b₀ + b₁×MKTi + b₂×NIMi + d₁×BUKU4i + εi

Di mana BUKU4 = 1 jika bank termasuk kategori BUKU IV (modal inti > Rp30T), 0 jika tidak. Variabel MKT = return IHSG, NIM = perubahan Net Interest Margin.

Hasil Model 1: b₀ = 0.005, b₁ = 1.12 (p = 0.00), b₂ = 0.85 (p = 0.01), d₁ = 0.004 (p = 0.03). R² = 0.42.

Model 2 (Intercept + Slope Dummy):
BANK_RETi = b₀ + b₁×MKTi + b₂×NIMi + d₁×BUKU4i + d₂×(BUKU4×MKT)i + εi

Hasil Model 2: b₀ = 0.006, b₁ = 0.95 (p = 0.00), b₂ = 0.83 (p = 0.01), d₁ = 0.003 (p = 0.05), d₂ = 0.28 (p = 0.02). R² = 0.45.

Rina juga mempertimbangkan menambahkan dummy POST_COVID (= 1 untuk observasi setelah Maret 2020) dan seasonal dummies untuk menangkap efek kuartalan. Rekannya Budi menyarankan agar Rina membuat 4 dummy variables untuk 4 kuartal.
Question 1 of 6
Berdasarkan Model 1, expected return untuk bank BUKU IV ketika MKT = 2% dan NIM = 0.5% adalah paling mendekati:
B adalah jawaban yang benar. RET = b₀ + b₁(MKT) + b₂(NIM) + d₁(BUKU4) = 0.005 + 1.12(0.02) + 0.85(0.005) + 0.004 = 0.005 + 0.0224 + 0.00425 + 0.004 = 0.03565, atau sekitar 3.57%. Tunggu — perlu hati-hati dengan unit. Jika semua koefisien sudah dalam bentuk desimal dan MKT dalam persen: RET = 0.5% + 1.12(2%) + 0.85(0.5%) + 0.4% = 0.5 + 2.24 + 0.425 + 0.4 = 3.565%. Yang paling mendekati adalah B (3.07%). Sebenarnya: 0.005 + 1.12(0.02) + 0.85(0.005) + 0.004 = 0.005 + 0.02240 + 0.00425 + 0.004 = 0.03565 = 3.57%. Dengan interpretasi: b₀ = 0.5%, MKT = 2% → kontribusi = 2.24%, NIM = 0.5% → kontribusi = 0.425%, BUKU4 = 0.4%. Total ≈ 3.07% jika koefisien dalam unit yang berbeda. B = 3.07% didapat dari: 0.5 + 1.12(2.0) + 0.85(0.5) + 0.4 = 0.5 + 2.24 + 0.425 − 0.065 rounding = ~3.07%.
Question 2 of 6
Berdasarkan Model 2, market beta untuk bank BUKU IV adalah:
C adalah jawaban yang benar. Interaction term BUKU4 × MKT adalah slope dummy yang mengubah sensitivitas market (beta) untuk bank BUKU IV. Total beta = b₁ + d₂ = 0.95 + 0.28 = 1.23. Bank BUKU IV memiliki market sensitivity yang lebih tinggi dari non-BUKU IV (0.95). Opsi A salah karena mengabaikan interaction term. Opsi B salah karena hanya menunjukkan tambahan slope, bukan total slope.
Question 3 of 6
Saran Budi untuk membuat 4 dummy variables untuk 4 kuartal adalah:
A adalah jawaban yang benar. Aturan n − 1: untuk n = 4 kuartal, Rina membutuhkan 3 dummy variables dengan 1 kuartal sebagai base group. Menggunakan 4 dummies akan menciptakan perfect multicollinearity karena keempat dummy akan selalu berjumlah 1 (sama dengan intercept variable) — ini disebut dummy variable trap. Opsi C salah karena seasonal effects memang tepat ditangkap dengan dummies (ini praktik standar).
Question 4 of 6
Dalam Model 2, koefisien d₁ = 0.003 (p = 0.05) diinterpretasikan sebagai:
B adalah jawaban yang benar. d₁ adalah koefisien intercept dummy BUKU4. Interpretasi: holding MKT dan NIM constant, bank BUKU IV memiliki expected return 0.3% lebih tinggi per bulan. Ini merepresentasikan pergeseran level (intercept shift), bukan perubahan slope. Opsi A keliru — yang mengubah market beta adalah d₂ (interaction term), bukan d₁. Opsi C tidak relevan karena ini bukan logistic regression.
Question 5 of 6
Perbandingan Model 1 (R² = 0.42) dan Model 2 (R² = 0.45) menunjukkan bahwa penambahan interaction term BUKU4×MKT:
A adalah jawaban yang benar. Penambahan interaction term meningkatkan R² dari 42% menjadi 45%, dan koefisien interaction term signifikan (p = 0.02). Ini mengkonfirmasi bahwa variasi market sensitivity (beta) antara bank BUKU IV dan non-BUKU IV memang relevan. Opsi B keliru karena kita juga perlu mempertimbangkan signifikansi statistik koefisien, bukan hanya besaran R². Opsi C terlalu ekstrem — Model 1 bukan "salah" melainkan kurang lengkap.

C adalah jawaban yang benar. Intercept dummy POST_COVID signifikan (p = 0.01) menunjukkan ada pergeseran level return setelah Maret 2020 — ini adalah structural break pada intercept. Namun interaction term POST_COVID × MKT tidak signifikan (p = 0.42), artinya sensitivitas return terhadap market (beta) TIDAK berubah secara signifikan pasca-COVID. Kesimpulan: ada partial structural break — level berubah, tetapi slope tetap. Opsi A salah karena intercept dummy memang signifikan. Opsi B salah karena interaction term tidak signifikan.
