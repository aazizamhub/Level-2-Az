Dummy Variables & Interaction Terms

Extensions of Multiple Regression — Subtopic 1 of 2

Learning Objectives — Hari ini lo harus bisa:
  • Definisikan dummy variable dan jelaskan kenapa kita butuh n − 1 dummies untuk n kategori
  • Bedakan intercept dummy vs slope dummy dan interpretasikan koefisiennya
  • Hitung efek interaction term pada dependent variable untuk tiap kategori

Theory · 30 min
1
Apa Itu Dummy Variable?

Dummy variable (atau indicator variable) adalah variabel yang bernilai 1 kalau suatu kondisi terpenuhi dan 0 kalau tidak. Gunanya: memasukkan qualitative information (kategori) ke dalam regression yang biasanya cuma handle angka kontinu.

Contoh: lo mau bedakan return saham berdasarkan sektor industri (Technology, Financial Services, Health Care, Energy, Food & Beverage). Lo bikin 4 dummy variables — satu kategori jadi base group (tanpa dummy).

Aturan: n kategori → butuh n − 1 dummy variables
// Base group = kategori tanpa dummy (koefisien dummy diinterpretasikan relatif terhadap base)
⚠ Exam Trap

Kalau lo pakai n dummies untuk n kategori (bukan n − 1), lo melanggar asumsi no exact linear relationship among X's — karena semua dummies akan sum to the intercept variable. Ini disebut dummy variable trap, dan regresi akan gagal.

2
Intercept Dummy vs Slope Dummy

Ada dua cara dummy variable mengubah regression line:

TipeModelEfek
Intercept Dummy Y = b₀ + d₀D + b₁X + ε Geser garis ke atas/bawah (parallel shift). D=0 → intercept b₀; D=1 → intercept (b₀ + d₀)
Slope Dummy Y = b₀ + b₁X + d₁(D×X) + ε Ubah kemiringan garis. D=0 → slope b₁; D=1 → slope (b₁ + d₁)
Both Y = b₀ + d₀D + b₁X + d₁(D×X) + ε Ubah intercept dan slope sekaligus
Visualisasi: Intercept dummy → dua garis paralel (slope sama, level beda). Slope dummy → dua garis dari titik awal yang sama tapi beda kemiringan. Both → dua garis yang beda level DAN kemiringan.
3
Interaction Terms — Slope Dummy in Action

Interaction term itu slope dummy yang tercipta dari perkalian dummy × continuous variable. Ini bikin efek suatu X terhadap Y berbeda-beda tergantung kategori.

Return = b₀ + b₁EXP + b₂CASH + b₃AGE + b₄SIZE + d₁BLEND + d₂GROWTH + d₃(AGE×BLEND) + d₄(AGE×GROWTH) + ε
// d₃ dan d₄ adalah interaction terms — slope dummies untuk AGE

Cara baca: Misal d₃(AGE_BLEND) = 0.005 (p = 0.53, not significant) dan d₄(AGE_GROWTH) = 0.020 (p = 0.01, significant). Artinya:

• Value funds (base): efek AGE = 0.065% per tahun

• Growth funds: efek AGE = 0.065 + 0.020 = 0.085% per tahun (significantly higher)

• Blend funds: efek AGE = 0.065 + 0.005 = 0.070% (not significantly different dari base)

⚠ Exam Trap

Jangan lupa: total slope untuk kategori D=1 = original slope + interaction coefficient. Banyak yang lupa menjumlahkan dan cuma pakai interaction coefficient saja — itu salah!

4
Structural Breaks & Seasonal Dummies

Dummy variables juga dipakai untuk mendeteksi structural breaks — perubahan mendadak dalam hubungan antar variabel akibat perubahan regulasi, krisis, atau regime change.

Cara kerja: Bikin dummy D = 1 untuk periode setelah event (misalnya setelah COVID, setelah regulasi baru). Masukkan sebagai intercept dummy dan/atau slope dummy. Kalau koefisien dummy signifikan → ada structural break.

Seasonal dummies: Kalau lo pakai data quarterly, bikin 3 dummies (Q1, Q2, Q3 — Q4 jadi base). Ini menangkap efek musiman, misalnya "January effect" atau seasonal earnings patterns.

Konteks Indonesia: Bayangkan lo bikin model return IHSG dengan dummy D = 1 untuk periode setelah tax amnesty (2016-2017). Kalau d₀ signifikan positif → intercept lebih tinggi post-tax amnesty, menunjukkan structural shift in the relationship.
Quick Reference Cheat Sheet
n categories → dummies

D=1 → intercept = b₀ + d₀
Slope Dummy
D=1 → slope = b₁ + d₁
Interaction Term

Practice — Item Set Vignette
Format CFA L2 · 4 soal · ~30 menit · Jawab dulu sebelum lihat penjelasan
Case Vignette
Dewi Kartika, CFA, adalah senior analyst di PT Mandala Capital, Jakarta. Ia membangun model regresi untuk menjelaskan return bulanan saham-saham sektor consumer di IDX menggunakan data 72 observasi. Model awalnya:

RETi = b₀ + b₁×MKTi + b₂×SIZEi + εi

Dewi ingin menguji apakah perusahaan halal-certified memiliki pola return yang berbeda. Ia menambahkan dummy variable HALAL (= 1 jika halal-certified, 0 jika tidak), serta interaction term HALAL×MKT. Model barunya:

RETi = b₀ + b₁×MKTi + b₂×SIZEi + d₁×HALALi + d₂×(HALAL×MKT)i + εi

Hasil estimasi: b₀ = 0.008, b₁ = 0.92, b₂ = −0.15, d₁ = 0.003 (p-value = 0.04), d₂ = 0.18 (p-value = 0.02).

Dewi juga ingin menguji apakah ada structural break setelah penerapan regulasi ESG baru pada Januari 2024. Ia menambahkan dummy POST = 1 untuk observasi setelah Jan 2024 dan menemukan koefisien POST = 0.005 (p-value = 0.35).
Question 1 of 4
Market sensitivity (beta terhadap MKT) untuk saham halal-certified berdasarkan model Dewi adalah:
B adalah jawaban yang benar. Interaction term HALAL×MKT adalah slope dummy — ia mengubah slope dari MKT ketika HALAL = 1. Total slope (beta) untuk halal stocks = b₁ + d₂ = 0.92 + 0.18 = 1.10. Opsi A salah karena 0.92 hanya berlaku untuk base group (non-halal). Opsi C salah karena d₂ hanyalah tambahan slope, bukan total slope.
Question 2 of 4
Koefisien d₁ = 0.003 pada variabel HALAL diinterpretasikan sebagai:
A adalah jawaban yang benar. HALAL adalah intercept dummy — koefisiennya (d₁ = 0.003) merepresentasikan selisih intercept antara halal dan non-halal. Artinya, holding other variables constant, saham halal memiliki expected return 0.3% lebih tinggi. Opsi B salah — ini bukan logistic regression, HALAL bukan dependent variable. Opsi C salah — yang mengubah slope (market sensitivity) adalah interaction term d₂, bukan d₁.
Question 3 of 4
Jika Dewi ingin menambahkan klasifikasi saham ke dalam 4 sektor (Consumer Staples, Consumer Discretionary, Healthcare, Technology), berapa dummy variable yang harus ditambahkan?
C adalah jawaban yang benar. Aturan fundamental: n kategori membutuhkan n − 1 dummy variables. Untuk 4 sektor, Dewi perlu 3 dummies — satu sektor menjadi base (reference) group, dan koefisien tiap dummy diinterpretasikan relatif terhadap base tersebut. Menggunakan 4 dummies (opsi A) akan menyebabkan perfect multicollinearity (dummy variable trap).
Question 4 of 4
Berdasarkan hasil dummy POST (koefisien = 0.005, p-value = 0.35), kesimpulan Dewi mengenai structural break setelah regulasi ESG adalah:
B adalah jawaban yang benar. Dengan p-value = 0.35 (jauh di atas significance level 5% maupun 10%), kita gagal menolak H₀ bahwa koefisien POST = 0. Artinya tidak ada bukti statistik bahwa regulasi ESG menyebabkan perubahan level return (structural break pada intercept). Opsi A salah karena mengabaikan p-value. Opsi C kurang tepat — meskipun slope dummy bisa memberikan informasi tambahan, intercept dummy test sudah cukup untuk menguji perubahan level.
