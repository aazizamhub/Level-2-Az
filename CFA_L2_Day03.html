Model Misspecification — 4 Jenis Kesalahan Functional Form

Sebelum masuk ke heteroskedasticity dan serial correlation, penting dipahami bahwa kedua masalah ini sering merupakan gejala dari model yang salah spesifikasi. Ada 4 jenis kegagalan functional form:

JenisPenjelasanKonsekuensi
Omitted variablesVariabel penting tidak dimasukkan ke modelHeteroskedasticity atau serial correlation
Wrong form of variablesHubungan non-linear tapi dimodelkan linearHeteroskedasticity
Inappropriate scalingData perlu di-scale/transform tapi tidak dilakukanHeteroskedasticity atau multicollinearity
Inappropriate poolingData dari regime berbeda digabung (pre/post krisis)Heteroskedasticity atau serial correlation
Key insight: Kalau lo menemukan heteroskedasticity atau serial correlation, jangan langsung "koreksi" dengan robust SE saja — tanyakan dulu: apakah ada model misspecification yang mendasari? Mungkin ada variabel penting yang terlewat atau bentuk fungsi yang salah.
2
Heteroskedasticity — Dua Tipe & Dampaknya

Heteroskedasticity terjadi ketika variance dari regression errors berbeda-beda antar observasi — melanggar asumsi homoskedasticity (Var(εᵢ) = σ² konstan).

TipeDefinisiDampak pada Inference
Unconditional Error variance tidak berkorelasi dengan independent variables Tidak masalah besar untuk statistical inference
Conditional Error variance berkorelasi dengan nilai independent variables Bermasalah — SE underestimated, t-stats inflated, Type I error ↑

Dampak conditional heteroskedasticity:

Coefficients masih unbiased — ini yang membedakan dari serial correlation (kalau ada lagged DV). Tapi standard errors underestimated → t-statistics inflated → kita cenderung menolak H₀ padahal seharusnya tidak (Type I error lebih banyak).

⚠ Exam Trap #1

Heteroskedasticity membuat coefficient estimates tetap unbiased tapi standard errors menjadi biased. Jangan balikkan — soal exam sering mencoba menjebak dengan mengatakan koefisiennya bias.

Breusch-Pagan (BP) Test — 3 Langkah:

Step 1: Estimasikan regression awal: Y = b₀ + b₁X₁ + b₂X₂ + u

Step 2: Regresi squared residuals terhadap semua independent variables: û² = a₀ + a₁X₁ + a₂X₂ + ε

Step 3: H₀: semua aⱼ = 0 (tidak ada conditional heteroskedasticity)

BP statistic = n × R² (dari Step 2) // Chi-square distributed, k degrees of freedom (k = jumlah X di Step 1) // One-tail right-side test | Tolak H₀ jika BP > critical chi-square
✅ Koreksi

Gunakan robust standard errors (White-corrected / heteroskedasticity-consistent standard errors). Koefisien tidak berubah — hanya SE yang dikoreksi menjadi lebih besar → t-stats turun → inference lebih valid.

3
Serial Correlation — Dampak & Deteksi

Serial correlation (autocorrelation) terjadi ketika regression errors berkorelasi antar observasi. Paling sering muncul pada data time series.

KondisiCoefficient EstimatesStandard Errors
Tidak ada lagged DV sebagai X Masih consistent Biased (underestimated)
Ada lagged DV sebagai X (Yₜ₋₁) Inconsistent (biased) Biased (underestimated)
⚠ Exam Trap #2 — Paling Sering Diuji

Ini adalah distingsi kritis di exam. Kalau tidak ada lagged dependent variable sebagai independent variable → serial correlation hanya merusak standard errors (coefficients masih oke). Kalau ada lagged DV → coefficients juga menjadi inconsistent/biased. Hafalkan tabel ini!

Positive serial correlation (paling umum): Residual positif di satu periode → residual periode berikutnya juga cenderung positif. Dampaknya sama dengan heteroskedasticity: SE underestimated → t-stats inflated → Type I error lebih banyak.

Breusch-Godfrey (BG) Test — lebih robust dari DW:

Step 1: Estimasikan regression awal → simpan residuals ûₜ

Step 2: Regresi ûₜ terhadap semua X original + lagged residuals: ûₜ = a₀ + a₁X₁ₜ + a₂X₂ₜ + ρ₁ûₜ₋₁ + eₜ

Step 3: H₀: ρ₁ = 0 (tidak ada serial correlation)

DW vs BG: Durbin-Watson hanya test first-order serial correlation dan tidak valid kalau ada lagged DV sebagai regressor. Breusch-Godfrey lebih robust — bisa detect serial correlation hingga lag-p dan valid meski ada lagged DV. Di exam, BG adalah preferred test.
✅ Koreksi

Gunakan Newey-West standard errors (HAC — Heteroskedasticity and Autocorrelation Consistent). Koreksi ini sekaligus menangani serial correlation DAN conditional heteroskedasticity — dua masalah dalam satu solusi.

Practice — Item Set Vignette
Format CFA L2 · 4 soal · ~30 menit
Case Vignette
Budi Santoso, CFA, adalah quantitative analyst di PT Bahana Sekuritas, Jakarta. Ia membangun dua model regression untuk klien institutional-nya:

Model Alpha: Regresi monthly excess return portofolio saham (PORT_RET) terhadap IHSG excess return (MKTRF), size factor (SMB), dan value factor (HML) menggunakan 120 bulan data. Model ini tidak menggunakan lagged PORT_RET sebagai independent variable.

Model Beta: Regresi quarterly GDP growth (ΔGDP) terhadap quarterly change in consumer spending (ΔCONS), menggunakan 60 kuartal data. Model Beta menggunakan ΔGDP dari kuartal sebelumnya (ΔGDPₜ₋₁) sebagai salah satu independent variable.

Hasil diagnostic tests:
— Model Alpha: BP test statistic = 13.40, p-value = 0.0038 (critical chi-square at 5% with k=3: 7.815)
— Model Alpha: BG test F-statistic = 1.85, p-value = 0.182 (critical F at 5%: 3.92)
— Model Beta: BG test F-statistic = 18.62, p-value = 0.0015 (critical F at 5%: 3.15)

Rekan Budi, Rina, menyarankan untuk menggunakan Durbin-Watson test pada Model Beta sebagai pengganti BG test karena "lebih sederhana."
Question 1 of 4
Berdasarkan hasil BP test pada Model Alpha, kesimpulan yang paling tepat adalah:
B adalah jawaban yang benar. BP test statistic 13.40 > critical chi-square 7.815 dengan p-value 0.0038 < 0.05 → tolak H₀ → ada conditional heteroskedasticity. H₀ BP test adalah "tidak ada conditional heteroskedasticity" (semua koefisien dalam regresi û² vs X sama dengan nol). Opsi A salah karena membandingkan nilai terbalik. Opsi C salah karena BP test adalah untuk heteroskedasticity, bukan serial correlation (serial correlation diuji dengan BG test).
Question 2 of 4
Mengingat Model Alpha terbukti memiliki conditional heteroskedasticity (tetapi tidak serial correlation), pernyataan manakah yang paling tepat mengenai dampaknya?
B adalah jawaban yang benar. Conditional heteroskedasticity tidak membuat coefficient estimates menjadi biased — coefficients masih unbiased (BLUE property hilang tapi unbiasedness tetap ada). Yang terpengaruh adalah standard errors: conditional heteroskedasticity menyebabkan SE underestimated → t-statistics inflated (terlalu besar) → kita terlalu sering menolak H₀ → Type I error meningkat. Koreksinya: gunakan robust/White-corrected standard errors. Opsi A salah karena heteroskedasticity tidak membuat coefficient inconsistent (itu masalah serial correlation dengan lagged DV). Opsi C salah — arahnya terbalik.
Question 3 of 4
Model Beta terbukti memiliki serial correlation berdasarkan BG test. Mengingat Model Beta menggunakan ΔGDPₜ₋₁ sebagai salah satu independent variable, dampak serial correlation pada Model Beta adalah:
C adalah jawaban yang benar. Ini adalah distingsi kritis: ketika ada lagged dependent variable (ΔGDPₜ₋₁) sebagai independent variable DAN ada serial correlation dalam error term, coefficient estimates menjadi inconsistent (biased bahkan dengan n → ∞). Ini jauh lebih serius daripada kasus tanpa lagged DV (di mana coefficients masih consistent). Dalam kasus ini, koreksi dengan robust SE saja tidak cukup — model perlu direvisi. Opsi A dan B salah karena mengabaikan implikasi keberadaan lagged DV.
Question 4 of 4
Saran Rina untuk menggunakan Durbin-Watson test sebagai pengganti BG test pada Model Beta adalah:
C adalah jawaban yang benar. Durbin-Watson test memiliki dua keterbatasan utama: (1) hanya menguji first-order serial correlation, dan (2) tidak valid ketika model menggunakan lagged dependent variable sebagai independent variable — yang persis adalah kondisi Model Beta. Breusch-Godfrey test jauh lebih robust: bisa mendeteksi serial correlation hingga lag-p yang ditentukan, dan tetap valid meski ada lagged DV. Opsi B justru membalik kenyataan — DW lebih terbatas (hanya first-order), BG yang bisa higher-order.
