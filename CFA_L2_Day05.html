Influence Analysis — High-Leverage vs Outlier

Ada dua jenis observasi yang bisa mempengaruhi regression results secara signifikan:

TipeDefinisiExtreme dalam halDideteksi dengan
High-Leverage Point Observasi dengan nilai X yang ekstrem — jauh dari rata-rata X Independent variable (X) Leverage hᵢᵢ
Outlier Observasi dengan nilai Y yang ekstrem — residualnya sangat besar Dependent variable (Y) Studentized Residual tᵢ*

Penting: Outlier dan high-leverage point tidak otomatis bermasalah. Yang bermasalah adalah ketika keduanya jauh dari regression line — menyebabkan garis regresi "tertarik" ke arah mereka dan mengubah slope coefficients.

Leverage (hᵢᵢ) — Rule of Thumb:

Leverage range: 0 ≤ hᵢᵢ ≤ 1
Potentially influential jika: hᵢᵢ > 3 × (k+1)/n // k = jumlah independent variables | n = jumlah observasi

Studentized Deleted Residual (tᵢ*) — Outlier Detection:

tᵢ* = eᵢ* / sₑ* ~ t(n−k−2) degrees of freedom // Flag sebagai outlier: |tᵢ*| > 3 // Flag sebagai potentially influential: |tᵢ*| > critical t(n−k−2)
⚠ Exam Trap

Degrees of freedom untuk studentized residual adalah n − k − 2, bukan n − k − 1 seperti pada t-test koefisien biasa. Satu degree of freedom tambahan hilang karena kita menghapus satu observasi saat menghitung residual. Sering jadi jebakan di soal.

💡 Apa yang Dilakukan Setelah Menemukan Influential Observation?

1. Cek apakah ada data entry error atau measurement error → koreksi atau hapus
2. Kalau datanya valid → mungkin ada omitted variable atau asumsi yang dilanggar
3. Bisa juga "winsorize" data — cap extreme values pada percentile tertentu

Dummy Variables — Intercept Dummy & Rule of n−1

Dummy variable (indicator variable) bernilai 1 jika kondisi tertentu terpenuhi, dan 0 jika tidak. Digunakan untuk memasukkan variabel kualitatif ke dalam regression.

⚠ Rule Paling Penting — n−1 Dummies

Untuk n kategori, gunakan n−1 dummy variables. Kategori yang tidak diberi dummy menjadi "base category" atau "control group". Kalau kita buat dummy untuk semua n kategori, akan terjadi perfect multicollinearity (dummy trap) karena semua dummy dijumlahkan = 1.

Contoh: 3 fund styles (Value, Blend, Growth) → 2 dummies (BLEND dan GROWTH). Value menjadi base. Atau: 12 bulan → 11 dummies. Desember menjadi base.

Intercept Dummy — Menggeser Intercept:

Yᵢ = b₀ + d₀D + b₁Xᵢ + εᵢ
D = 0 (base): Y = b₀ + b₁X
D = 1 (other category): Y = (b₀ + d₀) + b₁X
→ Garis paralel bergeser vertikal sebesar d₀

Interpretasi d₀: Selisih expected value of Y antara kategori D=1 vs base category D=0, holding X constant. Kalau d₀ = 2.50 (GROWTH), artinya growth funds menghasilkan return 2.50% lebih tinggi dari value funds (base), holding semua variabel lain constant.

Contoh dari CFA Curriculum: Model mutual fund returns dengan dummies BLEND (d₁=0.66) dan GROWTH (d₂=2.50), base = VALUE. Artinya: blend funds outperform value funds 0.66%/tahun, growth funds outperform value funds 2.50%/tahun, setelah mengontrol expense ratio, cash ratio, age, dan size.
3
Slope Dummy (Interaction Term) — Mengubah Slope

Kalau kita ingin slope hubungan X→Y berbeda untuk kelompok berbeda, gunakan slope dummy (interaction term antara D dan X):

Yᵢ = b₀ + b₁Xᵢ + d₁DᵢXᵢ + εᵢ
D = 0 (base): Y = b₀ + b₁X (slope = b₁)
D = 1 (other): Y = b₀ + (b₁ + d₁)X (slope = b₁ + d₁)

Bisa juga gabungkan intercept dan slope dummy sekaligus:

Yᵢ = b₀ + d₀D + b₁Xᵢ + d₁DᵢXᵢ + εᵢ
D = 0: Y = b₀ + b₁X
D = 1: Y = (b₀ + d₀) + (b₁ + d₁)X ← both intercept AND slope change
Contoh CFA — Mutual Fund dengan AGE Slope Dummy:
Model: Returns = b₀ + b₁EXP + b₂CASH + b₃AGE + b₄SIZE + d₁BLEND + d₂GROWTH + d₃AGE_BLEND + d₄AGE_GROWTH
AGE_GROWTH coefficient = 0.020 (significant, p=0.01). Artinya: untuk growth funds, setiap tambahan satu tahun usia fund, return naik (0.065 + 0.020) = 0.085% — lebih besar dari value funds (0.065%).
⚠ Slope Dummy ≠ Intercept Dummy

Intercept dummy → garis paralel (slope sama, intercept beda). Slope dummy → garis dengan kemiringan berbeda (slopes beda, intercept sama). Kombinasi keduanya → garis sepenuhnya berbeda (both differ). Soal exam sering menguji mana yang tepat untuk skenario tertentu.

Practice — Item Set Vignette
Format CFA L2 · 4 soal · ~30 menit
Case Vignette
Dewi Kurniawan, CFA, adalah quantitative analyst di PT Mandiri Sekuritas. Ia sedang membangun model cross-sectional untuk memprediksi return saham-saham IDX menggunakan dua independent variables: price-to-book ratio (PBV) dan debt-to-equity ratio (DER), dengan n = 80 observasi (k = 2).

Setelah mengestimasi model, Dewi menghitung leverage dan studentized residuals untuk semua observasi. Beberapa hasilnya:

Obs.hᵢᵢ (Leverage)Studentized Residual tᵢ*
Obs. 120.0821.842
Obs. 270.1480.213
Obs. 540.0613.412
Obs. 710.044−0.891

Critical t-statistic at 5% significance with n−k−2 = 75 degrees of freedom ≈ 1.992.

Selain itu, Dewi juga membangun model kedua yang memasukkan dummy variables untuk mengklasifikasikan saham ke dalam tiga sektor: Keuangan (Financial), Konsumer (Consumer), dan Industri (Industrial). Model kedua juga memasukkan interaction term antara sektor dan DER untuk menguji apakah pengaruh DER terhadap return berbeda per sektor.
Question 1 of 4
Berdasarkan rule of thumb leverage, berapa threshold hᵢᵢ untuk model Dewi (n=80, k=2), dan observasi mana yang potentially influential?
C adalah jawaban yang benar. Formula yang tepat adalah 3×(k+1)/n bukan 3×k/n. Dengan k=2 dan n=80: threshold = 3×(2+1)/80 = 3×3/80 = 9/80 = 0.1125. Dari empat observasi: Obs. 27 (hᵢᵢ = 0.148) melebihi 0.1125 → potentially influential. Obs. 12 (hᵢᵢ = 0.082) di bawah threshold. Opsi A dan B salah karena menggunakan 3×k/n (menghilangkan +1 untuk intercept). Perhatikan formula: k+1 karena leverage memperhitungkan intercept juga.
Question 2 of 4
Berdasarkan studentized residuals, observasi mana yang merupakan outlier dan/atau potentially influential?
B adalah jawaban yang benar. Dua langkah dalam memeriksa studentized residual: (1) |tᵢ*| > 3 → flag sebagai outlier. Obs. 54 dengan |t*| = 3.412 > 3 → ini adalah outlier. (2) |tᵢ*| > t-critical(n−k−2) → flag sebagai potentially influential. Dengan df = 80−2−2 = 76, t-critical ≈ 1.992. Obs. 54 juga melebihi ini → potentially influential. Opsi A salah karena leverage yang rendah tidak menghalangi sebuah observasi untuk influential berdasarkan studentized residual — keduanya adalah detectors yang berbeda (leverage untuk extreme X, studentized residual untuk extreme Y). Opsi C salah karena threshold outlier adalah |t*| > 3, bukan t-critical.
Question 3 of 4
Model kedua Dewi memasukkan dummy variables untuk tiga sektor (Financial, Consumer, Industrial). Berapa jumlah dummy variables yang tepat, dan sektor mana yang menjadi base category?
B adalah jawaban yang benar. Rule: untuk n kategori, gunakan n−1 = 3−1 = 2 dummy variables. Menggunakan 3 dummies untuk 3 kategori akan menyebabkan "dummy trap" — perfect multicollinearity karena ketiga dummy akan menjumlahkan ke variabel yang digunakan untuk estimasi intercept. Base category adalah kategori yang tidak diberi dummy. Pemilihan base category bersifat arbitrary — bisa Financial, Consumer, atau Industrial — yang penting hanya 2 dummy yang dibuat. Opsi C salah karena tidak ada aturan yang mengharuskan base category berdasarkan abjad.
Question 4 of 4
Model Dewi menghasilkan: Return = 0.50 − 0.30×FINANCIAL + 0.40×DER − 0.15×(FINANCIAL×DER). Untuk saham Financial, interpretasi yang paling tepat dari interaction term (FINANCIAL×DER) adalah:
C adalah jawaban yang benar. Ini adalah slope dummy (interaction term), bukan intercept dummy. Model: ketika FINANCIAL=1: Return = (0.50 − 0.30) + (0.40 − 0.15)×DER = 0.20 + 0.25×DER. Ketika FINANCIAL=0 (non-Financial): Return = 0.50 + 0.40×DER. Interaction term mengubah slope DER dari 0.40 (non-Financial) menjadi 0.25 (Financial) — perbedaan 0.15. Opsi A salah karena interaction term mengubah slope, bukan level. Opsi B salah karena koefisien total 0.25 masih positif dan signifikan — DER tetap berpengaruh, hanya slope-nya berbeda dari non-Financial.
