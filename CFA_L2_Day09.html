Theory · 30 min
1
Model Autoregressive: AR(1) dan AR(p)

Time-series analysis berbeda dari cross-sectional regression karena observasi punya urutan waktu — data hari ini tergantung data kemarin. Model autoregressive (AR) memanfaatkan hubungan ini: variabel diregresikan terhadap nilainya sendiri di masa lalu.

AR(1): xt = b₀ + b₁xt−1 + εt
// Nilai sekarang = intercept + slope × nilai kemarin + error
AR(p): xt = b₀ + b₁xt−1 + b₂xt−2 + ... + bpxt−p + εt
// p lag dipakai sebagai prediktor

Cara pilih order p: Mulai dari AR(1), cek autocorrelation residual. Kalau masih signifikan di lag tertentu, tambah lag → AR(2), dst. Pakai t-test pada residual autocorrelation: standard error = 1/√T. Kalau semua autocorrelation residual nggak signifikan, model sudah correctly specified.

Key Insight: Durbin-Watson statistic nggak valid untuk menguji serial correlation di AR model karena independent variable-nya adalah lagged dependent variable. Gunakan residual autocorrelation test saja.
2
Covariance Stationarity — Syarat Utama AR Model

Supaya AR model valid dan bisa diestimasi pakai OLS, time series harus covariance stationary. Kalau nggak? Hasil regresi jadi spurious — estimasi b₁ bias dan semua hypothesis test invalid.

SyaratPenjelasan
Mean konstan & finite E(xt) = μ untuk semua t, dan |μ| < ∞
Variance konstan & finite Var(xt) = σ² untuk semua t, dan σ² < ∞
Covariance konstan & finite Cov(xt, xt−s) = λs — hanya tergantung lag s, bukan waktu t

Contoh non-stationary: Data sales yang trending naik (mean nggak konstan), data dengan seasonality kuat, dan random walks (variance tak terbatas seiring waktu).

⚠ Exam Trap

Stationarity di masa lalu bukan jaminan stationarity di masa depan. Perubahan regime (misalnya krisis finansial, perubahan kebijakan moneter) bisa mengubah sifat time series secara fundamental — ini penting buat jawab soal tentang limitasi time-series forecasting.

3
Mean Reversion & Multiperiod Forecasting

Time series yang covariance stationary punya sifat mean reversion — kalau nilainya di atas rata-rata, model prediksi nilainya akan turun; kalau di bawah, model prediksi akan naik. Seperti termostat yang menjaga suhu ruangan.

Mean-reverting level: x̄ = b₀ / (1 − b₁)
// Hanya valid kalau |b₁| < 1 (covariance stationary)

Multiperiod forecast (chain rule): Untuk forecast 2 periode ke depan, substitusi forecast periode 1 ke dalam model:

x̂t+1 = b̂₀ + b̂₁xt
x̂t+2 = b̂₀ + b̂₁x̂t+1
// Semakin jauh forecast, semakin besar uncertainty
Key Insight: Semua multiperiod forecast dari AR model covariance stationary akan converge ke mean-reverting level seiring horizon makin panjang. Ini karena setiap iterasi menarik forecast lebih dekat ke b₀/(1 − b₁).
4
Random Walk & Unit Root Test (Dickey-Fuller)

Random walk adalah AR(1) dengan b₀ = 0 dan b₁ = 1 — artinya nilai terbaik untuk memprediksi x besok adalah x hari ini. Ini bukan covariance stationary karena variance-nya tumbuh tanpa batas seiring waktu.

Random walk: xt = xt−1 + εt
Random walk with drift: xt = b₀ + xt−1 + εt (b₀ ≠ 0)
// Mean-reverting level = b₀/(1−1) = undefined!

Dickey-Fuller (DF) test: Kita nggak bisa langsung test b₁ = 1 pakai t-test biasa (distribusinya invalid). DF test mentransformasi model:

xt − xt−1 = b₀ + g₁xt−1 + εt
// g₁ = (b₁ − 1). H₀: g₁ = 0 (unit root). Hₐ: g₁ < 0 (stationary)
// Pakai DF critical values, BUKAN t-table biasa!

First differencing: Kalau time series punya unit root, transformasi jadi yt = xt − xt−1. Series baru ini covariance stationary dan bisa dimodelkan pakai AR.

⚠ Exam Trap

R² yang sangat tinggi (misal 0.99) di AR(1) model bukan indikator model yang bagus — justru bisa jadi tanda random walk! Kalau lo first-difference dan R² jadi ~0, itu konfirmasi bahwa perubahan-perubahannya unpredictable (random walk).

5
Seasonality dalam Time-Series Models

Banyak data ekonomi/keuangan punya pola musiman (seasonal pattern). Misalnya, retail sales selalu tinggi di bulan Desember. Kalau nggak diatasi, seasonality bikin model misspecified — autocorrelation residual di lag musiman akan signifikan.

Deteksi: Cek autocorrelation residual di lag musiman (lag 4 untuk quarterly data, lag 12 untuk monthly). Kalau t-stat > critical value → ada seasonality.

Solusi: Tambahkan seasonal lag ke model. Contoh untuk monthly data:

xt = b₀ + b₁xt−1 + b₂xt−12 + εt
// Lag 12 menangkap pola tahunan di data bulanan
Key Insight: Menambahkan seasonal lag bisa secara dramatis meningkatkan R² dan menghilangkan serial correlation. Dalam contoh CFA curriculum, R² melonjak dari 0.05 ke 0.67 setelah menambahkan lag 12 pada data retail sales bulanan Kanada.
Quick Reference Cheat Sheet
AR(1) Model

Mean-Reverting Level
x̄ = b₀ / (1 − b₁)
Stationarity Condition
|b₁| < 1
Random Walk
xt = xt−1 + εt (b₁ = 1)
DF Test Hypothesis

Mini Vignette — PT Nusantara Semen Tbk
4 soal · AR modeling & stationarity
Vignette
Analis Dewi Puspita sedang membangun model time-series untuk memprediksi quarterly revenue growth PT Nusantara Semen Tbk, produsen semen terbesar di Indonesia. Data yang digunakan adalah 80 observasi kuartalan (Q1 2006 – Q4 2025). Dewi mengestimasi AR(1) model berikut:

Revenue growtht = 0.0248 + 0.6510 × Revenue growtht−1 + εt

Hasil statistik:
• R² = 0.4236, Standard error = 0.0312, Observations = 80
• t-stat intercept = 2.85 (p < 0.01), t-stat b₁ = 7.62 (p < 0.01)
• Residual autocorrelations: Lag 1 = 0.032 (t = 0.29), Lag 2 = −0.108 (t = −0.96), Lag 3 = 0.045 (t = 0.40), Lag 4 = 0.285 (t = 2.54)

Dewi juga mengestimasi AR(1) model untuk harga saham (level) PT Nusantara Semen dengan hasil: b̂₀ = 85.20, b̂₁ = 0.9945, R² = 0.9891. Dickey-Fuller test menghasilkan g₁ = −0.0055 dengan t-stat = −1.42 (DF critical value pada 5% = −2.88).
Soal 1 / 4
Berdasarkan residual autocorrelations dari AR(1) model revenue growth, model Dewi paling likely:
B adalah jawaban yang benar. Dengan T = 80 observasi, SE = 1/√80 = 0.1118. Critical value pada 5% ≈ 1.97. Lag 4 autocorrelation = 0.285 dengan t-stat = 2.54 > 1.97, sehingga signifikan. Ini menunjukkan seasonality (quarterly pattern) yang belum ditangkap. Solusinya bukan random walk (C salah karena model revenue growth punya b₁ = 0.651 < 1, bukan unit root), melainkan menambahkan seasonal lag (xt−4) ke model.
Soal 2 / 4
Mean-reverting level revenue growth kuartalan PT Nusantara Semen berdasarkan AR(1) model adalah paling mendekati:
A adalah jawaban yang benar. Mean-reverting level = b₀ / (1 − b₁) = 0.0248 / (1 − 0.6510) = 0.0248 / 0.3490 = 0.0711 atau 7.11%. Opsi B (2.48%) hanya intercept tanpa dibagi. Opsi C salah perhitungan. Jika revenue growth saat ini di bawah 7.11%, model memprediksi growth akan naik; jika di atas, akan turun.
Soal 3 / 4
Berdasarkan Dickey-Fuller test pada harga saham PT Nusantara Semen, kesimpulan yang paling tepat adalah:
C adalah jawaban yang benar. DF test: t-stat g₁ = −1.42. DF critical value pada 5% = −2.88. Karena −1.42 > −2.88 (tidak cukup negatif), kita gagal tolak H₀: g₁ = 0 (unit root ada). Ini berarti harga saham kemungkinan random walk dan harus di-first-difference sebelum bisa dimodelkan pakai AR. R² tinggi (0.9891) justru merupakan tanda random walk — bukan bukti model bagus (B salah). A salah karena kita gagal (bukan berhasil) menolak H₀.
Soal 4 / 4
Jika revenue growth kuartal ini (Q4 2025) adalah 5.50%, forecast revenue growth untuk Q2 2026 (dua periode ke depan) menggunakan chain rule of forecasting paling mendekati:
B adalah jawaban yang benar. Step 1 — Forecast Q1 2026: x̂t+1 = 0.0248 + 0.6510(0.0550) = 0.0248 + 0.0358 = 0.0606 (6.06%). Step 2 — Forecast Q2 2026: x̂t+2 = 0.0248 + 0.6510(0.0606) = 0.0248 + 0.0395 = 0.0643 ≈ 6.43%. Paling mendekati jawaban B (6.35%). Catatan: forecast bergerak menuju mean-reverting level 7.11% (C) tapi belum sampai di horizon 2 periode. A (5.94%) hanya satu periode ke depan yang dibulatkan salah.
